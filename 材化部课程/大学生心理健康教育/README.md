# 大学生心理健康教育

> 本文件为该课程文件夹的资料导航。完整学习建议见 [大学生心理健康教育.md](大学生心理健康教育.md)。

## 内容介绍

大学生心理健康教育（俗称"大心"）是面向全校本科生的公共基础/通识必修课，通常安排在大一学年修读。课程性质偏向"水课"，以心理健康常识普及与心理调适方法为主，学业压力较小。

22 级由王静老师授课，老师人品很好，课堂氛围轻松，经常让大家开怀大笑。期末考核为随堂撰写一篇大作文，主题由老师自定，只要认真完成写作，给分一般不低于 90 分。整体而言只需按时到课、完成期末作文即可获得较高分数，无需额外复习。

# 无机化学实验

> 本文件为该课程文件夹的资料导航。完整学习建议见 [无机化学实验.md](无机化学实验.md)。

## 内容介绍

无机化学实验是大一年级第一门实验课，属材化部大类基础课，公共实验室位于生农化楼 701 三楼。实验较为简单、基本无危险，考试为实验操作，正常上课一般不会挂科。22 级可选教师较多，不同老师风格差异较大，选对老师直接影响实验报告工作量与给分。

各老师特点：李宝龙（和蔼、事少，推荐）、李红喜（脾气暴躁但报告可二合一，反感抄书与忤逆）、袁亚仙（人好负责但更辛苦）、胡传江（强化班专属，事少负责看得开）、李亚红（强化班专属，负责、须严格按步骤）、任志刚（最严格、脾气大、给分最低）、苏远停（报告最少、给分较高）、张文华（非 ban 必选，报告少给分最高）、贾定先（拔尖班专属，严格中规中矩）。本文件夹附进度表、线上考试说明与装置图等实验资料。

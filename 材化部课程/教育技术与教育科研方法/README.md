## 内容介绍

教育技术与教育科研方法是面向师范方向学生的师范方向必修/专业基础课，通常在本科中段修读。课程融合教育技术应用与教育科研方法论两大模块，内容涵盖教学媒体与信息化教学设计、教育研究的基本流程与常用方法（如文献研究、调查研究、实验研究、个案研究等），旨在培养师范生开展教育科研与信息化教学的能力。

本文件夹暂无独立学习建议文档（无 `.md` 文件），仅提供课程 Word 资料，可作为复习与作业参考。

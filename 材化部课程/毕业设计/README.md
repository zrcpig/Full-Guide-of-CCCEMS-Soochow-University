# 材料与化学化工学部毕业设计

材化部毕业设计主要时间是在大四整个学年，需要提前联系毕业导师。

## 格式问题

学位论文的格式是最让人头疼的部分，因此我在试用了 LaTex (Overleaf, VS Code, KeepResearch)和 Word (2024)后，得出结论：

**LaTex**：目前Github及Gitee上的（截至2026年）苏州大学LaTex模板并未完全按照学校要求编写，重新写一个本科生学位论文LaTex模板工程量较大，暂且没有心力去完成这项任务（寄希望于后来人了）。

<details>
<summary>目前LaTex编写平台及编写模板问题</summary>

| LaTex编写平台 | 中文书籍推荐 |
| :-------: | :-------: |
| VS Code | 本地配置难度高，需要有编程基础，目前最为推荐的方式 |
| Overleaf | 昂贵，免费版的编译时间太短，不适合写毕业论文 |
| KeepResearch | 苏大图书馆推荐方式，库支持有问题，只能使用预设的库（？），自己导入库时有报错，对bibtex支持性差 |

</details>

**Word**：仍然是本科生学位论文最佳的编写方式，为方便后来者编写学位论文，我封装了一个 .dotx 模板，请自行下载取用。

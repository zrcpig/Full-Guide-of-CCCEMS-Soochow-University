# 材料与化学化工学部毕业设计

材化部毕业设计主要时间是在大四整个学年，需要提前联系毕业导师。

## 格式问题

学位论文的格式是最让人头疼的部分，因此我在试用了 LaTex (Overleaf, VS Code, KeepResearch)和 Word (2024)后，得出结论：

**LaTex**：目前Github及Gitee上的（截至2026年）苏州大学LaTex模板并未完全按照学校要求编写，重新写一个本科生学位论文LaTex模板工程量较大，暂且没有心力去完成这项任务（寄希望于后来人了）。

**Word**：仍然是本科生学位论文最佳的编写方式吗，为方便后来者编写学位论文，我封装了一个 .dotx 模板，请自行下载取用。

# 思想政治理论课实践

> 本文件为该课程文件夹的资料导航。完整学习建议见 [思想政治理论课实践.md](思想政治理论课实践.md)。

## 内容介绍

思想政治理论课实践是面向全校本科生的公共基础必修实践环节，通常安排在大一到大二的暑假进行，仅 0.5 学分但须严格按学校要求完成，否则照样会挂科。实践形式分为校内实践团与自主联系单位两类。

校内实践方面，材化部内有许多实践团，内容涵盖去陕西支教、苏州支教、工厂参观等，虽有面试但招人较多，无需担心；材化部实践团给的志愿时长较为丰厚，若能获得媒体报道还可"超级加倍"。自主实践方面，可自行联系单位，往年优秀案例为两位学姐暑期赴清华大学研学并盖到清华公章；也可找单位盖章后撰写（水够字数的）实践报告手抄上交。

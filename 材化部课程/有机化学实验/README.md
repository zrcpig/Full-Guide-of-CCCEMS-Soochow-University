# 有机化学实验

> 本文件为该课程文件夹的资料导航。完整学习建议见 [有机化学实验.md](有机化学实验.md)。

## 内容介绍

有机化学实验贯穿整个学年，单次实验时间很长，经常需要做一整天。22 级可选教师有汪顺义和袁丹：汪顺义老师"非 ban 必选"，但有机上实验的操作考试给分较低；袁丹老师较严格，上课很负责，也可选择。

本文件夹收录了约 24 个经典有机实验的教案（如对甲基苯乙酮、呋喃甲酸、苯甲酸重结晶、环己烯、溴乙烷、醇酚性质、醛酮性质、己二酸、三苯甲醇、乙酰乙酸乙酯、聚乙烯醇缩甲醛、咖啡因提取等），以及 13 级张杨飞学长的复习资料、操作考试资料和七年制复习与思考题答案，供预习、复习操作考试使用。

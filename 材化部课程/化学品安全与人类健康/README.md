# 化学品安全与人类健康

> 本文件为该课程文件夹的资料导航。完整学习建议见 [化学品安全与人类健康.md](化学品安全与人类健康.md)。

## 内容介绍

化学品安全与人类健康是化学类专业的一门专业选修/通识选修课，聚焦实验室化学品安全、危险化学品管理与人体健康防护等内容。课程在培养方案中作为安全教育与专业拓展类课程，对从事化学实验与科研的学生具有实际指导意义。

主文档记录该课由学部徐小平老师讲授。徐老师兼任学部安全管理员，课上常以学部实验室真实安全案例（如四氯硅酸未盖盖触发烟雾传感器、漏水漏到实验室等）穿插讲解，内容相对有意思。课程整体偏水，可在课上做自己的事，偶尔听听案例也有助于增强实验室安全意识。本文件夹另收录高校实验室安全检查项目表，可作为安全规范参考。

# 苏州大学材料与化学化工学部完全指南

> 养天地正气，法古今完人。

这是一篇苏州大学材化部的本科学习指南，也是对自己大学三年学习生涯的一个纪念。

本指南由苏大材化 22 级学长编写并维护，可以帮助学弟学妹快速了解材化部、高效完成材化部本科专业的学习。如有补充与建议，欢迎评论或邮件 *heyurui@alu.suda.edu.cn*。

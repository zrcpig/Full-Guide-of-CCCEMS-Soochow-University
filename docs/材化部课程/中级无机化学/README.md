# 中级无机化学

> 化学类专业专业选修课，在本科无机化学基础上深入讨论配位化学、固体结构、过渡金属化学等专题。

## 学习建议

- 由李宝龙老师讲授（22 级），讲课比较催眠，点名形式主义（课前发纸签到，可代写）。
- 课堂听讲与平时作业要求都不高，作业可基本忽略。
- 期中前会发一份期中试题，不限制答题工具与时间，但需签到签卷（迟到有"弱者"标记）。
- **分数大头由期末论文决定**，李老师重点关注论文格式，务必严格按照**苏州大学毕业论文标准格式**撰写，并逐项满足其要求。
- 库里收录的期末论文仅供参考，***不要照抄***，否则一切后果自负。
- 总而言之，这是专业选修中**给分较高的水课**。

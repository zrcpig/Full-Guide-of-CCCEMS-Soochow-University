# 中级无机化学

> 本文件为该课程文件夹的资料导航。完整学习建议见 [中级无机化学.md](中级无机化学.md)。

## 内容介绍

中级无机化学是化学类专业的一门专业选修课，一般在本科中高年级开设，在培养方案中作为无机化学方向的进阶与拓展课程。课程在本科无机化学基础上，深入讨论配位化学、固体结构、过渡金属化学等专题内容。

主文档记录该课由李宝龙老师讲授（22 级），其讲课偏催眠、点名较为形式化（课前发纸签到，可代写）。课程对课堂听讲与平时作业要求都不高，作业可基本忽略；期中前会发一份期中试题，不限制答题工具与时间，但需注意签到签卷。

课程分数大头由**期末论文**决定，李老师重点关注论文格式，因此务必严格按照**苏州大学毕业论文标准格式**撰写，并逐项满足其要求。主文档明确提醒：库里收录的期末论文仅供参考，切勿照抄。总体而言，该课是专业选修中给分较高的水课。

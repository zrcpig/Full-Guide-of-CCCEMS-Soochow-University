# 中国近现代史纲要

> 本文件为该课程文件夹的资料导航。完整学习建议见 [中国近现代史纲要.md](中国近现代史纲要.md)。

## 内容介绍

中国近现代史纲要（简称"近代史"或"史纲"）是高校思想政治理论课的重要组成部分，属于面向全体本科生的公共必修课，一般安排在大一上学期。课程系统讲述自近代以来中国谋求民族独立与人民解放的奋斗历程，帮助学生理解历史和人民怎样选择了中国共产党、选择了社会主义道路。

授课方面，主文档提到该课由马克思主义学院的胡绿叶老师讲授，讲课比较有意思但整体偏"水"，听课与否可自行把握。平时考核包含小组拍 vlog 与小组汇报，给分不低，且因开设于大一上学期，小组活动有助于同学之间熟悉、拉近感情。

期末考试全部为主观题，"会扯就有分"，建议把教材后面的思考题背一下作为答题素材。本文件夹收录的期末资料（思考题、期末考试、近代史答案终极版）正是针对主观题备考的实用材料。

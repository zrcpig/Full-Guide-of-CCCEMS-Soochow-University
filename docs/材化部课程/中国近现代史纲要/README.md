# 中国近现代史纲要

> 高校思想政治理论课的重要组成部分，系统讲述近代以来中国谋求民族独立与人民解放的奋斗历程。

## 学习建议

- 面向全体本科生的公共必修课，一般安排在大一上学期。
- 由马克思主义学院[胡绿叶](https://marxism.suda.edu.cn/22/87/c30845a467591/page.htm)老师讲授，上课比较水，讲课有意思，想听的可以听听。
- 小组大作业为拍 vlog 与小组汇报，给分不低；大一上学期开课，有助于同学之间熟悉、拉近感情。
- 期末考试全部为主观题，"会扯就有分"，建议把教材后面的思考题背一下作为答题素材。
- 本文件夹收录的期末资料（思考题、期末考试、近代史答案终极版）是针对主观题备考的实用材料。

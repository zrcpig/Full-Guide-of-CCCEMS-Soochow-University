# 化学品安全与人类健康

> 聚焦实验室化学品安全、危险化学品管理与人体健康防护的专业选修/通识选修课。

## 学习建议

- 由学部[徐小平](https://chemistry.suda.edu.cn/31/80/c31982a405888/page.htm)老师讲授，比较水，但内容相对有意思。
- 徐老师兼任学部安全管理员，课上常以学部实验室真实安全案例穿插讲解：
  - 研究生做实验未盖四氯硅酸盖子，大量烟雾触发传感器，徐老师凌晨披大衣赶来排查。
  - 研究生未关水龙头，漏水漏到徐老师实验室。
- 课上可以做自己的事，偶尔听听案例有助于增强实验室安全意识。
- 本文件夹收录的高校实验室安全检查项目表可作为安全规范参考。

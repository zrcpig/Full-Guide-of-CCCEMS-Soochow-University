# 现代有机合成新技术

> 现代有机合成新技术是材化部赵应声老师开设的专业选修课，是专业选修课里最极端的一门——给分标准完全由赵老师心情决定。

## 学习建议

- 这门课由材化部赵应声老师讲授。
- 专业选修课里最极端的一门课，给分标准完全由赵老师的心情决定。
- 赵老师特别喜欢点名，且毫无规律。曾在一堂课上连点三次名（第一节课下课、第二节课下课、第三节课课间突击点名）。只要有逃课习惯，这门课就是雷课。点中一次少 20 分。
- 上课可以完全不听，做自己的事情就好，但不要发出声音。
- 期末作业是完成一篇以**苏州大学毕业论文标准格式**为格式的论文，建议记下讲课 PPT 的标题，方便找灵感。
- **水和雷并存的一门课，有逃课习惯这门课就是雷课**。

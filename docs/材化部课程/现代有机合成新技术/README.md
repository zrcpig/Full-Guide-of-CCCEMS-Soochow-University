# 现代有机合成新技术

> 本文件为该课程文件夹的资料导航。完整学习建议见 [现代有机合成新技术.md](现代有机合成新技术.md)。

## 内容介绍

现代有机合成新技术是材化部赵应声老师开设的专业选修课，是专业选修课里最极端的一门——给分标准完全由赵老师心情决定。

赵老师特别喜欢点名，且毫无规律，点中一次少 20 分，曾在一堂课上连点三次名（第一节课下课、第二节课下课、第三节课课间突击）。只要有逃课习惯，这门课就是雷课。上课可完全不听，自己做自己的事即可，但不要发出声音。

期末作业是完成一篇以**苏州大学毕业论文标准格式**为格式的论文，建议记下讲课 PPT 的标题以便找灵感。总体是"水和雷并存"的一门课。

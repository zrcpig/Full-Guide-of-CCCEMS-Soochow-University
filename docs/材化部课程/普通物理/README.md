# 普通物理

> 本文件为该课程文件夹的资料导航。完整学习建议见 [普通物理.md](普通物理.md)。

## 内容介绍

普通物理是材化部第二门容易挂科的课程，开设于大一下学期（普物上）和大二上学期（普物下），属于培养方案中的学科基础必修课。

22 级普物上/下由朱天淳老师（现已退休）授课，是普物教得最好、给分最高、作业和课堂事务最少的老师。普物上主要学习麦克斯韦方程组和动力学等内容，多数同学高中未系统接触，需要大量刷题，尤其是麦克斯韦方程组部分；普物下主要学习光学、热学，更考验理解能力，看不懂就刷题，实在不懂要及时问老师。

考核方面，期中试卷由各讲课老师自行批改本班试卷，因此不同班级均分可能差异较大；期末试卷整体难度不高，但由多位老师打乱统改、给分较严，需注意书写规范。复习应以刷题为主，重点攻克麦克斯韦方程组与光学、热学典型题型。

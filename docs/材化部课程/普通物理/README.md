# 普通物理

> 材化部第二门容易挂科的课程，开设于大一下学期（普物上）和大二上学期（普物下），属培养方案中的学科基础必修课。

## 学习建议

给我上普物上/下的老师是朱天淳（目前已退休）老师，是普物教得最好、给分最高、作业和课堂事情最少的老师。

普物上主要学麦克斯韦方程组和动力学这些，很多人高中都没学过，所以需要大量刷题，尤其是麦克斯韦方程组部分。

普物下主要学光学、热学，主要考验理解能力，看不懂就刷题，实在不懂问老师。

普物的期中试卷是由讲课的各个老师判改自己班的试卷，所以会出现较水老师的班期中平均分比其它班高出一大截这样。

普物的期末试卷基本不会很难，但是判卷是几个老师打乱了判，给分比较严格，注意书写规范。

复习应以刷题为主，重点攻克麦克斯韦方程组与光学、热学典型题型。

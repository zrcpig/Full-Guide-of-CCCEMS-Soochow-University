# 思想道德与法治

> 本文件为该课程文件夹的资料导航。完整学习建议见 [思想道德与法治.md](思想道德与法治.md)。

## 内容介绍

思想道德与法治是大一下学期的思想政治理论课（公共基础必修），整体偏"水课"，课堂以理论讲授为主，无需投入过多精力。22 级由张洋阳老师授课，上课不管学生做自己的事，但会无形中给一定压力。

考核方面，期末全部为主观题，考前半小时背诵"思考题全集"即可应对。无需平时跟进听课，把精力留给专业课，考前突击思考题是性价比最高的复习策略。

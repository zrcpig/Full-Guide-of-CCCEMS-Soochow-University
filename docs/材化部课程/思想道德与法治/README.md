# 思想道德与法治

> 大一下学期的思想政治理论课（公共基础必修），课堂以理论讲授为主，整体偏轻松。

## 学习建议

- 水课，无需平时跟进听课，把精力留给专业课。
- 22 级由张洋阳老师授课，上课不会管学生做什么，但会无形中给一定压力。
- 期末全部为主观题，考前半小时背诵"思考题全集"即可应对，这是性价比最高的复习策略。

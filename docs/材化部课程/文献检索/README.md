# 文献检索

> 本文件为该课程文件夹的资料导航。完整学习建议见 [文献检索.md](文献检索.md)。

## 内容介绍

文献检索是面向材化部本科生的大类基础/专业基础课，通常在本科中段修读。课程内容涵盖常用数据库检索、文献管理、信息素养等。该课"雷"与"水"取决于选到的老师，选老师直接决定课程体验。

姚英明老师（材化部副主任）对课内事务关心较少，考核形式为小组汇报，给分较高，属"水课"；沈风雷老师则布置多种耗时作业、常点名，最后一节为闭卷考试且试卷得分普遍不高、并直接影响最终成绩，属"大雷课"。建议未选到合适老师时该课可暂缓修读。

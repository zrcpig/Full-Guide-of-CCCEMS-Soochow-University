# 文献检索

> 面向材化部本科生的大类基础/专业基础课，涵盖常用数据库检索、文献管理、信息素养等。

## 学习建议

- **选老师直接决定课程体验**：
  - **姚英明老师**（材化部副主任）：对课内事务关心较少，考核为小组汇报，给分较高，属**水课**。
  - **沈风雷老师**：布置各种浪费时间的作业，常点名，最后一节为闭卷考试且试卷得分普遍不高、直接影响最终成绩，属**大雷课**。
- 建议没选到合适老师时这门课可以先不学。

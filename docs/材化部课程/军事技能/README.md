## 内容介绍

军事技能即大学新生军训，是面向全体本科生的公共必修实践环节，通常在新生入学后集中进行。课程内容包括队列训练、内务整理、军事常识普及等，考核以出勤与实训表现为准，旨在增强国防观念与组织纪律性。

主文档仅一句话说明"这门课就是军训，详见苏州大学军训"，未展开更多学习建议。本文件夹当前仅含主文档，未收录额外的军训资料。

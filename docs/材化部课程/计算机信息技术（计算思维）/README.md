# 计算机信息技术（计算思维）

> 本文件为该课程文件夹的资料导航。完整学习建议见 [计算机信息技术（计算思维）.md](计算机信息技术（计算思维）.md)。

## 内容介绍

计算机信息技术（计算思维）以 C++ 编程为主，附带计算机原理等基础内容，由黄蔚老师讲授，推荐选其课程。第一次过程化考试为开卷，考查基础知识；第二次过程化考试需多写 C 程序并巩固基础。

22 级期末采用苏州大学考试系统线上进行。大作业成品已收入本库，可供参考但请尊重知识产权、勿照抄。资料含大作业成品与过程化考试说明。

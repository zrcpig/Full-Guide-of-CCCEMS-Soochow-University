# 计算机信息技术（计算思维）

> 以 C++ 编程为主、附带计算机原理等基础内容的课程，黄蔚老师授课推荐。

## 学习建议

- 推荐选黄蔚老师，上课负责，不会因小事上纲上线，经常和学生打成一片。
- 第一次过程化考试为开卷，考查基础知识，可把题库抄书上。
- 第二次过程化考试需要多写 C 编程，基础知识平时多看看，不会太难。
- 22 级期末采用苏州大学考试系统线上进行。
- 大作业成品已收入本仓库，请参考但不要照抄，尊重知识产权。

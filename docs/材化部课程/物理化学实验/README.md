# 物理化学实验

> 与仪器分析实验类似的实验课，考试采用抽号方式到对应老师处做实验操作。

## 学习建议

物理化学实验与仪器分析实验类似，没有固定老师，考试是抽号去对应老师那里做实验操作，学期初会有笔试，考智慧树上的选择题和实验原理。

本文件夹按上、下学期分别收录了实验安排表、各实验 PPT、实验报告与思考题，供预习、撰写报告和准备笔试使用。

## 文件夹架构

```
物理化学实验/
├── 物理化学实验上/
│   ├── 2024-2025秋-第一学期-物化实验安排.pdf
│   ├── 物化实验上PPT/
│   ├── 物化实验上实验报告/
│   └── 物化实验上思考题/
└── 物理化学实验下/
    ├── 物化下实验安排汇总表.docx
    ├── 物化实验下PPT/
    └── 物化实验下实验报告/
```

## 资料说明

| 子目录/文件 | 用途 |
| :--- | :--- |
| `物理化学实验上/` | 第一学期（上）实验资料 |
| `2024-2025秋-第一学期-物化实验安排.pdf` | 2024-2025 秋季学期物化实验安排表 |
| `物化实验上PPT/` | 上学期各实验讲义 PPT，用于预习 |
| `物化实验上实验报告/` | 上学期实验报告参考 |
| `物化实验上思考题/` | 上学期各实验思考题，应对学期初笔试 |
| `物理化学实验下/` | 第二学期（下）实验资料 |
| `物化下实验安排汇总表.docx` | 下学期实验安排汇总表 |
| `物化实验下PPT/`、`实验报告/` | 下学期各实验讲义 PPT 与实验报告参考 |

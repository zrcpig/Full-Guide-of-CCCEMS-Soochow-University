## 内容介绍

马克思主义基本原理（简称马原）为公共必修思想政治理论课。期末考试全部为主观题，建议考前半小时翻阅课本课后思考题以便现场组织作答。小组作业为 PPT 汇报。

资料含复习思考题、复习纲要与复习资料，便于主观题背诵与梳理。

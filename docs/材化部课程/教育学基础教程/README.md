## 内容介绍

教育学基础教程（教育学原理）是面向师范方向学生的师范方向必修/专业基础课，通常在本科中段修读，是教师教育模块的核心理论课程。课程内容涵盖教育的本质、教育目的、教育制度、课程与教学、教师与学生等基本原理，部分班级使用项贤明版教材。

本文件夹暂无独立学习建议文档（无 `.md` 文件），仅提供教育学原理课后练习题及其参考答案（项贤明版），可用于课后练习与考前自测。

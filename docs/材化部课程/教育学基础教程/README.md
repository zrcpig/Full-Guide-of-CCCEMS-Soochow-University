# 教育学基础教程

> 面向师范方向学生的专业基础课，是教师教育模块的核心理论课程。

## 学习建议

教育学基础教程（教育学原理）通常在本科中段修读。课程内容涵盖教育的本质、教育目的、教育制度、课程与教学、教师与学生等基本原理，部分班级使用项贤明版教材。

本文件夹提供教育学原理课后练习题及其参考答案（项贤明版），可用于课后练习与考前自测。

# 材料化学

> 本文件为该课程文件夹的资料导航。完整学习建议见 [材料化学.md](材料化学.md)。

## 内容介绍

材料化学是材化部的专业选修课。22 级由王会芳老师授课，王老师性格负责（若选修樊建芬老师的《量子化学基础》，常能看到王老师坐第一排认真听课）。

该课程在专业选修中属于相对轻松的课，考核方式为小组 PPT 汇报，理论上抱保研生大腿即可不必担心汇报。但考勤较严格（22 级是最后一届支持雨课堂远程签到），课上还不定时进行智力小测验，保证出勤可有 80 分保底；其余分数需争取担任小组组长，并在给王老师的邮件中客观陈述个人贡献。本文件夹收录了王会芳老师各章 PPT（绪论、材料结构、材料性能、金属材料、无机非金属材料），供汇报与复习参考。

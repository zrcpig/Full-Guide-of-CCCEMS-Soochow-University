# 材料化学

> 材料化学是材化部的专业选修课，22 级由王会芳老师授课，该课程在专业选修中属于相对轻松的一门课。

## 学习建议

- 22 级上课的老师是王会芳老师，王老师性格比较负责。如果选樊建芬老师的《量子化学基础》课程，你会经常看到王老师拿个小本本聚精会神坐第一排听课。
- 这门课在专业选修中属于"水课"，考核方式是做小组 PPT，理论上只要抱保研生的大腿，小组汇报就完全不用担心。
- 考勤较为严格，22 级是最后一届能以雨课堂远程签到的一届，课上还不定时会有各种智力小测验。保证出勤的话保底有 80 分。
- 剩下的分数需要争当小组组长，然后在给王老师的邮件里客观陈述你的贡献即可。
- 课程 PPT 涵盖绪论、材料结构、材料性能、金属材料、无机非金属材料等内容，供汇报与复习参考。

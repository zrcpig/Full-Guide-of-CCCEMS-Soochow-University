# 中国特色文化英语教学

> 以中国特色文化为载体的特化型**英语阅读**课，兼顾文化自信培养与英语能力提升。

## 学习建议

- 属于公共基础/通识性质的英语课程，通常面向大一新生开设。
- 课程围绕中国传统文化、当代国情等内容展开英语阅读与表达训练。
- 建议结合本文件夹内的 MOOC 期末资料与教材答案进行复习：
  - MOOC 期末 PDF 可用于了解线上考核的题型与考查范围。
  - 教材答案方便核对课后习题，巩固阅读理解与词汇表达。

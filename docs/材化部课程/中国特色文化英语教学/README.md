# 中国特色文化英语教学

> 本文件为该课程文件夹的资料导航。完整学习建议见 [中国特色文化英语教学.md](中国特色文化英语教学.md)。

## 内容介绍

中国特色文化英语教学是一门以中国特色文化为载体的特化型**英语阅读**课，属于公共基础/通识性质的英语课程，通常面向大一新生开设。课程围绕中国传统文化、当代国情等内容展开英语阅读与表达训练，兼顾文化自信培养与英语能力提升。

主文档仅以一句话点明课程性质（特化型英语阅读课），未展开具体授课老师与复习要点。建议结合本文件夹内的 MOOC 期末资料与教材答案进行复习：MOOC 期末 PDF 可用于了解线上考核的题型与考查范围，教材答案则方便核对课后习题、巩固阅读理解与词汇表达。

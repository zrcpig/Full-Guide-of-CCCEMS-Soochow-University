# 摄影艺术欣赏与创作

> 本文件为该课程文件夹的资料导航。完整学习建议见 [摄影艺术欣赏与创作.md](摄影艺术欣赏与创作.md)。

## 内容介绍

摄影艺术欣赏与创作是出名的有趣通识选修课，面向全校本科生。由王岩（男）与王文利（女）两位老师共同授课：王岩老师主讲摄影理论并带队实操采景，王文利老师主要负责课堂气氛与微信群互动。

授课安排上，前一节为理论课，在教室讲解摄影基本原理（理论课上尽量认真听，以免被坐最后一排的王文利老师点名批评）；后一节为实操课，王岩老师带队在苏大校园各处采景，趣味性强，且实操课不点名。给分十分痛快，整体人均 98 分左右，若作品常被王岩老师在课上点评，则有望拿到 100。本文件夹附有大作业参考样例，可供创作思路参考。

# 摄影艺术欣赏与创作

> 出名的有趣通识选修课，面向全校本科生，由王岩与王文利两位老师共同授课。

## 学习建议

- 两位老师风格不同：王岩（男）主讲摄影理论并带队实操采景；王文利（女）主要负责课堂气氛与微信群互动。
- **理论课**（前一节）：在教室讲解摄影基本原理，尽量认真听，王文利老师坐最后一排，课上干其他事会被点名批评。
- **实操课**（后一节）：王岩老师带队在苏大校园各处采景，趣味性强，实操课不点名，可以学到非常实用的摄影技巧。
- **给分**：整体人均 98 分左右，王岩老师给分特别痛快；若作品常被放课上点评，分数很有可能是 100。
- 本文件夹附有大作业参考样例，可供创作思路借鉴。

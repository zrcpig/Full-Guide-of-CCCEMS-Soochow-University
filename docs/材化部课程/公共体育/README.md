# 公共体育

> 本文件为该课程文件夹的资料导航。完整学习建议见 [公共体育.md](公共体育.md)。

## 内容介绍

公共体育是面向全体本科生的公共必修课，贯穿本科各学期，学生可自选体育项目。培养方案中作为公共基础课程，与体测、评奖评优挂钩。课程一般由平时出勤、技能测试与期末理论考试等环节组成，正常参加考试即可通过。

主文档指出：公共体育课项目随便选，正常参加考试基本不会挂科；但需注意 2025 版评奖评优要求中，"三好学生"荣誉称号需要体育成绩达到 **B 等级以上**，因此每学期的 1000 米/2400 米跑要尽力完成。大三大四学期，若引体向上做不了，拿 B 会比较困难（除非跳远等其它项目特别突出）。期末理论考试有专用题库与考生指南，本文件夹收录的资料即针对该理论考试。

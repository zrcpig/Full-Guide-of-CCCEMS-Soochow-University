# 跨文化交际

> 特化型英语阅读类课程，陈莉莉老师授课负责且善解人意。

## 学习建议

- 陈莉莉老师授课中规中矩，比较负责并善解人意。
- 大作业为小组汇报。
- 期中期末考试会考查一些较偏门的文化交际知识，内容基本出自课本，考前需翻阅翻译并记忆大概。
- **重点刷 MOOC 题目**，反复刷题是备考关键。

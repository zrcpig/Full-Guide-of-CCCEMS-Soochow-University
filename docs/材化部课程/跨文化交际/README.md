# 跨文化交际

> 本文件为该课程文件夹的资料导航。完整学习建议见 [跨文化交际.md](跨文化交际.md)。

## 内容介绍

跨文化交际属特化型英语阅读类课程，由陈莉莉老师讲授，授课负责且善解人意。大作业为小组汇报。

期中期末考试会考查一些较偏门的文化交际知识，内容基本出自课本，考前需翻阅翻译并记忆大概。备考重点为 MOOC 题目，建议反复刷题。资料含各单元思维导图、MOOC 真题与教材 Case Study、课程安排及陈莉莉老师 PPT。

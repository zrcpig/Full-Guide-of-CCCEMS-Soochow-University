# 物理化学

> 需少量高数基础的学科基础核心课，开设于大二上（物化上）和大二下（物化下）。

## 学习建议

物理化学需要一点点高数基础，尤其是物化上，需要推导公式，用到一些微积分的知识。

物化能选的老师有孙如和徐敏敏：

- **孙如老师**：出考研卷的老师，上课普遍难，而且给分不算很高。
- **徐敏敏老师**：非 ban 必选，神中神，六边形老师，善解人意，讲得好，给分高，材化部为数不多满足"不可能三角"的老师。

物化的精髓就在于刷题。

物化上用的教材是南大编的《物理化学》第六版（比武大编的分析写得好太多了），内容很详实，随书会有本紫色的习题解答（上下册内容均有），建议收一下二手。平时的话就是多刷习题解答上的题，不会的多翻翻书，然后就能应付苏大的考试了。

物下同理，由于这个时间段很多人考研，不会来上物化下的课，因此很多人考得不高，多用心就能拿高分。

本文件夹另含 635 物理化学考研资料，便于下放题目练习与考研复习。

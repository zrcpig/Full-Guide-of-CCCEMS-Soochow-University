# 物理化学

> 本文件为该课程文件夹的资料导航。完整学习建议见 [物理化学.md](物理化学.md)。

## 内容介绍

物理化学需少量高数基础，尤其物化上需要推导公式、用到一些微积分知识，开设于大二上（物化上）和大二下（物化下），属培养方案中的学科基础核心课。

可选教师有孙如和徐敏敏：孙如老师是出考研卷的老师，上课普遍较难，给分不算很高；徐敏敏老师"非 ban 必选"，讲得好、善解人意、给分高，是材化部为数不多满足"不可能三角"的老师。

物化的精髓在于刷题。物化上教材为南大编的物化第六版（内容详实，比武大版分析写得好），随书有紫色习题解答（上下册内容均有），建议收二手；平时多刷习题解答上的题、不会的多翻书即可应对苏大考试。物化下同理，因该时段很多人考研不来上课，考得普遍不高，多用心即可拿高分。本文件夹另含 635 物理化学考研资料，便于下放题目练习与考研复习。

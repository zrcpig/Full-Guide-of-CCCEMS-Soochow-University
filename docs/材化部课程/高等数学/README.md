# 高等数学

> 本文件为该课程文件夹的资料导航。完整学习建议见 [高等数学.md](高等数学.md)。

## 内容介绍

高等数学是材化部大一年级难度最高、学分占比极大的核心基础课。上学期由徐娟娟老师讲授（2022 年苏大数学系博士毕业），下学期由王承富老师讲授，两位老师均负责且答疑积极。

教材采用苏大版高等数学，配套习题册是学习核心（约 70% 期中原题出自其上，其余 30% 为教材例题），无需另刷其他教辅。22 级下学期因难度过高发生「挂科事件」，可见该课对大学四年影响深远，务必选好老师并自学、复习、多刷题。资料按上下学期分目录，含各老师 PPT、期中期末真题与练习册。

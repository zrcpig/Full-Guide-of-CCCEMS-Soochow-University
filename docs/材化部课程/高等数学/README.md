# 高等数学

> 材化部大一年级难度最高、学分占比极大的核心基础课。

## 学习建议

高等数学是材化部大一年级难度最高的课程，而且在整个材化部培养方案中学分占比极大，**非常重要的课**。

高等数学上的老师是徐娟娟老师（2022 年刚从苏大数学系博士毕业），高等数学下的老师是王承富老师。两个老师都很负责，有问必答那种。

高等数学这门课用的教材是苏大出版的《高等数学》，这本书配套会有一个习题册（**非常重要**），之前这本书会发，但从 22 级开始没有了。因此想学好高等数学一定要刷这本习题册（70% 的期中考试原题就在上面），以及数学书上的例题（剩下 30% 的题目），除此以外别的书都不需要刷了。

22 级的高等数学是最难的一届，臭名昭著的"高等数学下挂科事件"就是我们这一届，当时挂掉了材化部 55% 以上的同学，导致很多人因为高等数学直接大学四年摆烂了，影响极其深远。

高等数学的成绩对大学生涯影响极大，选个好老师（推荐上面这两位），以及一定要自学、复习、多刷题——得高等数学者得大一。

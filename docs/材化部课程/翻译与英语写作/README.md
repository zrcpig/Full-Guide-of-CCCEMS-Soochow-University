# 翻译与英语写作

> 学校特化型英语写作课，目标之一是辅助六级写作与翻译训练。

## 学习建议

- 课程内容偏西方文学理论与英语写作"八股"，对理科生实用性有限，可以不重点听。
- 老师可能会布置与课堂内容关联不大的作业，还会课上点名抽查。
- 考试与日常授课关联不大，主要考查两篇小作文和翻译（类似浙江高考风格）。
- 翻译踩点得分，重准确而非文采（信达雅非必需）。
- 平时多积累词汇，22 级考试翻译主题为苏州戏剧，考试主题不确定，词汇量是应对关键。

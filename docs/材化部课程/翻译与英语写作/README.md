# 翻译与英语写作

> 本文件为该课程文件夹的资料导航。完整学习建议见 [翻译与英语写作.md](翻译与英语写作.md)。

## 内容介绍

翻译与英语写作是学校特化型英语写作课，目标之一是辅助六级写作与翻译训练。课程内容偏西方文学理论与英语写作「八股」，对理科生实用性有限。

考试与日常授课关联不大，主要考查两篇小作文和翻译（类似浙江高考风格）：翻译踩点得分，重准确而非文采（信达雅非必需）。22 级考试翻译主题涉及苏州戏剧，建议平时多积累词汇以应对不同主题。资料含约 22 份作文/翻译教学文档及期末题型说明。

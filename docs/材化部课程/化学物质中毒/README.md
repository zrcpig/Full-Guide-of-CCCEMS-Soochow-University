# 化学物质中毒

> 化学类专业专业选修/通识选修课，由医学部老师讲授，聚焦化学物质的毒性机制、中毒途径、临床表现与防护救治等内容。

## 学习建议

- **水课**，由医学部张增利老师讲授（22 级为网课）。
- 课程考核形式为小组汇报，傍个好队友，给分不会低于 90 分。
- 本文件夹收录的剧毒化学品、危化品申请表、精麻药品等资料，可作为小组汇报与了解危化品管理的参考。

## 文件夹架构

```
化学物质中毒/
└── 课程资料/
    ├── 剧毒化学品.docx
    ├── 危化品申请表.docx
    └── 精麻药品.docx
```

## 资料说明

- `课程资料/`：危化品管理相关课程资料。
  - `剧毒化学品.docx`：剧毒化学品名录/管理资料，小组汇报参考。
  - `危化品申请表.docx`：危险化学品申购/管理表样，了解危化品采购流程。
  - `精麻药品.docx`：精神药品与麻醉药品管理资料，了解特殊药品管制规定。

# 化学物质中毒

> 本文件为该课程文件夹的资料导航。完整学习建议见 [化学物质中毒.md](化学物质中毒.md)。

## 内容介绍

化学物质中毒是化学类专业的一门专业选修/通识选修课，由医学部老师讲授，聚焦化学物质的毒性机制、中毒途径、临床表现与防护救治等内容。课程在培养方案中作为安全健康与专业拓展类课程，对从事化学实验的学生了解化学品毒理与防护有实际价值。

主文档记录该课由医学部张增利老师讲授（22 级为网课），课程偏水，考核形式为小组汇报，傍个好队友即给分不会低于 90 分。本文件夹另收录剧毒化学品、危化品申请表、精麻药品等课程资料，可作为汇报与了解危化品管理的参考。

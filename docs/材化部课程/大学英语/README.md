## 内容介绍

大学英语是面向全校非英语专业本科生的公共基础必修课，通常按大一上、下两学期修读，并按入学英语分级（普通班/提高班等）进行分层次教学。课程涵盖听力、阅读、写作、翻译等模块，以《大学英语学习指导与能力训练》为主干教材与配套练习。

本文件夹暂无独立学习建议文档（无 `.md` 文件），仅提供《大学英语学习指导与能力训练》的答案与听力文本 PDF，可供课后对答案与听力练习跟读使用。

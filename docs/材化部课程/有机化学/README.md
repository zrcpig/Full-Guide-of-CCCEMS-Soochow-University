# 有机化学

> 比较容易挂科的一门学位课，开设于大一下学期（有机上）和大二上学期（有机下），属培养方案中的学科基础核心课。

## 学习建议

有机化学学习起来比较吃天赋，需要多做题（达到考研水平）才能拿高分。

22 级有机化学可选教师有赵蓓、孙宏枚、袁丹、葛健锋：

- **赵蓓老师**：苏大本科毕业，教授有机化学双语，上课会用英语 PPT（考试试卷也是英文，试题内容与中文班一致），但讲课会用中文，讲得比较好。
- **孙宏枚老师**：浙大毕业，教有机化学中文班，讲得很好，性格有点刀子嘴豆腐心，很照顾学生。
- **袁丹老师**：强化班专属，讲课较难，对挂科的学生会"格外照顾"。
- **葛健锋老师**：材料班、转专业重修班专属，讲得比较简单，不愿意随便挂人。

建议就是多刷题，不仅要刷课内的课本，还要刷苏大考研题——很多期末考试题都来自于考研题下放。

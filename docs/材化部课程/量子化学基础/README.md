# 量子化学基础

> 本文件为该课程文件夹的资料导航。本文件夹暂无独立学习建议文档，资料以教材、课件、思考题与期中真题为主。

## 内容介绍

量子化学基础是材化部高年级专业课程，与结构化学（一）衔接，深入讲解量子力学在化学中的应用。从资料文件名推断，课程主教材为黄明宝著《量子化学教程》（中国科学院大学版），樊建芬老师提供配套课件与每周思考题。

课件内容覆盖薛定谔方程、算符代数、对易子、角动量、氢原子、变分法与微扰、电子自旋与泡利原理、多电子波函数、Hartree-Fock 从头计算、密度泛函理论及量子化学计算软件等主题，并附 2018 与 2025 两版课件及相应期中/期末复习提纲。考核含期中考试（2022 级真题已收录）及每周思考题。

> 本文件夹暂无独立学习建议文档，以上简介依据资料文件名合理推断，供参考。

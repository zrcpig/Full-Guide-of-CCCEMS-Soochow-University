# 结构化学（一）

> 结构化学（一）是材化部大三阶段难度较高的专业核心课之一，符号表达带有"苏大特色"，需以课堂与本校 PPT/作业为准。

## 学习建议

- 这门课有两个老师可以选（22 级）：王会芳老师和樊建芬老师，两位老师授课风格相近，单纯看 PPT 又很难看懂，是大三比较难的一门课。
- 上课要认真听老师讲课，尤其是需要大四选「量子化学基础」的同学，本课是其重要基础。
- 很多符号的表达有"苏大特色"，B 站网课应付不了考试，需以课堂和本校 PPT/作业为准。
- 期中考试相对简单，基本考查 PPT 中的基本概念，多看 PPT，多刷作业题（尤其是樊建芬老师的作业题，期中考试大概率原题复现）。
- 期末考试较难，实在不会写的不要空，尽力写一些就有分；想追求高分还是要反反复复背 PPT 和作业题。
- 文档里有两个 mp3 文件是 2025 年期末考试前两位老师强调的复习范围，可供参考。

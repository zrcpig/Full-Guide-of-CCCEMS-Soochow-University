# 结构化学（一）

> 本文件为该课程文件夹的资料导航。完整学习建议见 [结构化学（一）.md](结构化学（一）.md)。

## 内容介绍

结构化学（一）是材化部大三阶段难度较高的专业核心课之一，也是后续选修「量子化学基础」的基础。22 级由樊建芬老师和王会芳老师并行开课，两位老师授课风格相近，课堂内容以 PPT 为主，但单纯看 PPT 较难自学。

该课符号表达带有「苏大特色」，B 站等通用网课难以覆盖考试要求，需以课堂与本校 PPT/作业为准。期中考试相对简单，基本考查 PPT 中的基本概念与作业题（樊建芬老师作业题大概率原题复现）；期末考试较难，建议反复背诵 PPT 与作业题，不会写的题目尽量作答争取步骤分。

参考资料以两位老师各章课件、期中/总复习提纲、各级期中期末真题为主，另附课后答案及 2025 年期末复习范围录音（mp3）。

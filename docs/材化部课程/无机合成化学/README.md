# 无机合成化学

> 本文件为该课程文件夹的资料导航。完整学习建议见 [无机合成化学.md](无机合成化学.md)。

## 内容介绍

无机合成化学是材化部专业方向课程，22 级由张文华老师讲授。平时上课较水、很少查出勤（偶尔点名），考勤分给得较高，但**整体给分较低**。课程有期末考试与大作业：大作业约 2000 字即可拿满，早交有额外加成。

最关键的是期末考试——**会考 PPT 里的内容，且考得较难**。22 级期末时张老师曾说最后一节 PPT 不用看（均为了解内容），但实际试卷中至少 30 分考的正是最后一节 PPT。因此复习时务必**通读全部章节课件**，不可轻信"了解内容不用看"的说法。本文件夹收录张文华老师的全部 PPT，含英文拓展阅读、作业汇总与第 1-6 章分章课件。

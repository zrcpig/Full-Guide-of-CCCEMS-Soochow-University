# 无机合成化学

> 材化部专业方向课程，22 级由张文华老师讲授。

## 学习建议

- 平时上课较水，很少查出勤（偶尔点名），考勤分给得较高。
- **整体给分较低**，需注意。
- **大作业**：约 2000 字即可拿满，最好早点交，有额外加成。
- **期末考试**：会考 PPT 里的内容，且考得较难。22 级期末时张老师称最后一节 PPT 不用看（均为了解内容），但实际试卷中至少 30 分考的正是最后一节 PPT。
- **复习策略**：务必**通读全部章节课件**，不可轻信"了解内容不用看"的说法。

## 资料说明

- `张文华老师的PPT/`（英文拓展阅读）：Chapter 1-6 英文拓展阅读资料，含 PDF、docx 及 Schlenk 线操作视频，用于拓展专业英语与实验技能。
- `张文华老师的PPT/`（作业汇总）：多次课后作业汇总（pptx/pdf），便于核对作业。
- `张文华老师的PPT/`（第 1-6 章课件）：分章课件 PDF（无机合成进展、气体与溶剂、无水无氧操作、结晶重结晶、柱层析、配位化合物合成、无机纳米材料），**期末复习须通读全部章节**。

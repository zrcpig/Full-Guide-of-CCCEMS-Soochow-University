# 普通物理实验

> 本文件为该课程文件夹的资料导航。完整学习建议见 [普通物理实验.md](普通物理实验.md)。

## 内容介绍

普通物理实验是配合普通物理课程开设的实验课，平时分占比较高，且上课地点经常变动，务必在课前确认下一节课的教室位置。

教材基本无用，每节课都有对应的讲义，照着讲义操作即可完成实验。本文件夹收录了普通班与强化班的实验报告参考以及历届实验报告存档，供预习和撰写报告时参考。

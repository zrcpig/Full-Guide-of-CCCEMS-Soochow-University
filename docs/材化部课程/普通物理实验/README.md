# 普通物理实验

> 普通物理实验是配合普通物理课程开设的实验课，平时分占比较高，且上课地点经常变动，务必在课前确认下一节课的教室位置。

## 学习建议

- 这门课平时分占比较高，而且上课地点经常变动，一定要在课前搞清楚下一节课教室在哪里。
- 教材几乎没有用，因为每节课都有对应的讲义，照着讲义来做就好。
- 本文件夹收录了普通班与强化班的实验报告参考以及历届实验报告存档，供预习和撰写报告时参考。

## 文件夹架构

```
普通物理实验/
├── 普物实验报告/
│   └── 2022级实验报告/
└── 普物实验资料/
    ├── 强化班实验报告.pdf
    └── 普通班实验报告.pdf
```

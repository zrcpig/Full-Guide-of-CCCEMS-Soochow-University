# 无机化学实验

> 大一年级第一门实验课，属材化部大类基础课，公共实验室位于生农化楼 701 三楼。

## 学习建议

无机实验比较简单，基本没有危险，考试为实验操作，正常上课的这门课没有挂的。

22 级可选教师较多，不同老师风格差异较大，选对老师直接影响实验报告工作量与给分：

- **李宝龙老师**：和蔼可亲的小老头，课堂事情比较少，推荐。
- **李红喜老师**：脾气较为暴躁，但课堂实验报告写的可以比较少（指预习报告/实验报告二合一），非常反感你直接抄书，非常反感你忤逆他的指导。
- **袁亚仙老师**：人脾气非常好，也非常负责，所以你上她的课会比别的老师更辛苦一点。
- **胡传江老师**：强化班专属，非 ban 必选的好老师，事少负责看得开。
- **李亚红老师**：强化班专属，实验室在胡老师隔壁的老师，同时教化学专业英语，人很负责，所以请严格按实验步骤来，不要耍花招。
- **任志刚老师**：刚子，特别负责的老师，可能会让你对实验课产生心理阴影的那种，要求是所有老师里最严格的，脾气也是，给分是最低的一档。
- **苏远停老师**：中规中矩，实验报告写的最少，给分比较高。
- **张文华老师**：非 ban 必选的好老师，脾气好，报告写的少，给分还最高。
- **贾定先老师**：拔尖班专属，严格的老师，中规中矩。

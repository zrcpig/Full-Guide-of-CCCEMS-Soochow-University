# 英语报刊选读

> 本文件为该课程文件夹的资料导航。完整学习建议见 [英语报刊选读.md](英语报刊选读.md)。

## 内容介绍

英语报刊选读是特化型英语阅读课程，主要为六级阅读做准备。夏天老师负责且体谅学生，作业少，遇到枯燥内容会主动建议学生自行安排时间。

期中期末考试均为阅读理解，题量较大、多为文章阅读，其中大部分为课内文章，建议通读课内各篇翻译。资料含各周作业样例及期末试卷说明。

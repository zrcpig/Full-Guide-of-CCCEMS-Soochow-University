# 英语报刊选读

> 特化型英语阅读课程，主要为六级阅读做准备，夏天老师授课负责且体谅学生。

## 学习建议

- 推荐选夏天老师，为人负责，作业少，性格好，能体谅学生难处。
- 课上遇到枯燥内容老师会主动建议学生自行安排时间，不强迫学习。
- 期中期末考试均为阅读理解，试题册很厚，多为文章阅读。
- 大部分为课内文章，建议通读课内各篇文章的翻译。

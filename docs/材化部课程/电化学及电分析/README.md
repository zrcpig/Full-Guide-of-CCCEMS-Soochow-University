# 电化学及电分析

> 本文件为该课程文件夹的资料导航。完整学习建议见 [电化学及电分析.md](电化学及电分析.md)。

## 内容介绍

电化学及电分析由副校长姚建林老师（厦门大学田中群院士开山大弟子）授课（22 级），姚老师非常负责，对课上的内容很上心。

考勤主要靠点名，大多数情况在上课后几分钟点名，只要不缺勤，这门课不会低于 85 分。此外会布置各种来源于姚老师即兴想法的"作业"——22 级第一次大作业即要求用 Python 写一个能以实际实验数据完美拟合巴-伏方程的程序。"期末考试"为小组汇报。总体属专业选修里比较水、给分很高的一门课。本文件夹收录了大作业与小组作业参考。

# 电化学及电分析

> 电化学及电分析由副校长姚建林老师授课，姚老师非常负责，考勤靠点名，不缺勤即可保证 85 分。

## 学习建议

- 这门课由副校长姚建林（厦门大学田中群院士开山大弟子）老师讲课（22 级），姚老师非常负责，对课上的内容很上心。
- 考勤主要靠点名，大多数情况在上课后几分钟点名，只要不缺勤，这门课不会低于 85 分。
- 课程会布置各种来源于姚老师即兴想法的"作业"，22 级第一次大作业即要求用 Python 写一个程序，能用实际实验数据完美拟合巴-伏方程，作业参考已收录。
- "期末考试"为小组汇报。
- 总体来说算是专业选修里比较水的，**给分很高**。

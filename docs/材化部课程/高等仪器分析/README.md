# 高等仪器分析

> 本文件为该课程文件夹的资料导航。完整学习建议见 [高等仪器分析.md](高等仪器分析.md)。

## 内容介绍

高等仪器分析由李建国老师讲授，整体给分宽松，属专业选修中较「水」的课程。学习无需深究课堂讲授，略读 PPT 并熟记期中题库与期末补充题库即可取得 90 分以上。

课件覆盖绪论、紫外可见分光光度新方法、催化动力学分析法、分子发光分析法、化学发光与电致化学发光等章节。资料含李建国老师 PPT、综合复习题、南京大学仪器分析习题答案及多届期中复习资料。

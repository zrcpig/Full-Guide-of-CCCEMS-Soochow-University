# 无机化学

> 大一年级第一门学位课程，属材化部大类基础课，分上、下两学期开设。

## 学习建议

无机化学不难，但拿高分难，这门课挂科率不高。

22 级无机化学的授课老师有袁亚仙、任志刚、胡传江（强化班专属）、贾定先（拔尖班专属）四位：

- **袁亚仙老师**：上课很负责，对学生很慷慨，曾在 23 级请全班喝咖啡，而且对学生的问题知无不答（各种问题，甚至袁老师还会关心你是不是谈恋爱）。
- **任志刚老师**：刚子，头像是只蓝色兔子，材化部工会主席，上课很严肃，但讲的课废话比较少，知识点比较直接，有很简单的不会的问题问他有可能会收获他的嘲笑。
- **胡传江老师**：强化班专属，非 ban 必选，给分高脾气好作业少，也会耐心解答你的问题。
- **贾定先老师**：拔尖班专属，刚子强化版，出的题更难，讲课扩展的知识很多，废话也很多。

无机化学上下想要学好就是得多做题。**不建议**使用材化部郎建平老师主编的那本大部头无机，**推荐**直接看吉林大学宋天佑老师的蓝皮《无机化学》（是材化部无机的超集），随蓝皮无机配套有一本无机化学习题，要全部刷过去。

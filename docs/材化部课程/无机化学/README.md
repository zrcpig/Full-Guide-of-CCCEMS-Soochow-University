# 无机化学

> 本文件为该课程文件夹的资料导航。完整学习建议见 [无机化学.md](无机化学.md)。

## 内容介绍

无机化学是大一年级第一门学位课程，属材化部大类基础课，分上、下两学期开设。课程不难但拿高分难，挂科率不高。22 级授课老师有袁亚仙、任志刚、胡传江（强化班专属）、贾定先（拔尖班专属）四位：袁老师负责慷慨、知无不答；任（刚子）老师严肃直接、知识点不绕弯；胡老师给分高脾气好；贾老师为"刚子强化版"，题目更难、扩展多。

学习方法上，关键在多做题。**不建议**使用材化部郎建平老师主编的大部头无机教材，**推荐**直接使用吉林大学宋天佑老师的蓝皮《无机化学》（是材化部无机的超集），并配合其配套习题集全部刷过去。本文件夹按上、下学期组织，含任志刚老师（上学期）与袁亚仙老师（下学期）的课件 PPT，以及历年资料与小组作业。

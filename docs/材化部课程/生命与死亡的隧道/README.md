# 生命与死亡的隧道

> 本文件为该课程文件夹的资料导航。完整学习建议见 [生命与死亡的隧道.md](生命与死亡的隧道.md)。

## 内容介绍

生命与死亡的隧道由医学部[何玉龙](https://sklrmp.suda.edu.cn/40/50/c18155a409680/page.htm)老师讲授，主要讲授癌症与血液病——苏州大学血液病研究全国第一，认真听讲可深入了解相关知识。

课程为小班授课，助教每节课挨个点人数，不存在逃课不被发现的可能；但何老师相当 nice，最后给分没有低于 90 的。考核形式为翻译文献（何教授在最后一节课布置几篇文献供翻译，翻出大意即可，越详细给分越高）以及两人一组的小组汇报，注意内容避免明显科学错误，何教授会耐心指正，类似提前培训开组会。本文件夹收录了小组作业参考与 22 级期末考试论文参考。

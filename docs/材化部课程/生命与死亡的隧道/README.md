# 生命与死亡的隧道

> 生命与死亡的隧道由医学部何玉龙老师讲授，主要讲授癌症与血液病相关知识，小班授课，给分慷慨。

## 学习建议

- 这门课由医学部[何玉龙](https://sklrmp.suda.edu.cn/40/50/c18155a409680/page.htm)老师讲授，主要讲癌症和血液病。苏州大学的血液病研究是全国第一，认真听的话确实能对血液病相关知识有所深入了 解。
- 小班授课，助教每节课挨个点人数，不存在逃课不被发现的可能。
- 何老师相当 nice，最后给分没有低于 90 的。
- 考核形式为翻译文献：何教授会在最后一节课上提前布置几篇文献让你翻译，翻译大概意思即可，越详细给分越高。
- 小组汇报为两人一组，注意内容不要有太显著的科学错误，何教授都会耐心给你指正，**类似提前培训你开组会**。

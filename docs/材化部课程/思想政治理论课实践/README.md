# 思想政治理论课实践

> 面向全校本科生的公共基础必修实践环节，通常安排在大一到二暑假进行，仅 0.5 学分但须按学校要求完成。

## 学习建议

- 千万要看清楚学校要求，虽然只有 0.5 学分但不按要求来照样会挂科。
- **校内实践**：材化部内有许多实践团，内容包括去陕西支教、苏州支教、工厂参观等；虽然会组织面试但招的人多，无需担心。材化部实践团给的志愿时长较为丰厚，若能获得媒体报道志愿时长还能超级加倍。也可参加其它学院的实践团。
- **自主实践**：可自行联系单位做实践，材化部往年优秀案例为两位学姐暑期赴清华大学研学，最终盖到了清华大学公章。也可以找单位盖一个章，撰写一篇实践报告（**一定要水够字数**）手写抄上去。

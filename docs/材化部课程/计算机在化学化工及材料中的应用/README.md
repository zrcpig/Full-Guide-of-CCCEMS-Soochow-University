# 计算机在化学化工及材料中的应用

> 面向化学专业的应用类课程，可获取 Origin、ChemDraw、Photoshop 等专业常用软件学习版。

## 学习建议

- 22 级由杨甫老师讲授（已离职），课程可获取各种化学专业必须的软件学习版，如 Origin、ChemDraw、Photoshop 等。
- 给分不算高，期末考试较难，22 级为线上考试且题目更难，考试成绩直接计入课程成绩。
- 建议可以选高年级的跨专业选修以减轻压力。

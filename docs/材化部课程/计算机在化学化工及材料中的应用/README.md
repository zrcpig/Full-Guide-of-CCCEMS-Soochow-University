# 计算机在化学化工及材料中的应用

> 本文件为该课程文件夹的资料导航。完整学习建议见 [计算机在化学化工及材料中的应用.md](计算机在化学化工及材料中的应用.md)。

## 内容介绍

计算机在化学化工及材料中的应用是面向化学专业的应用类课程，22 级由杨甫老师讲授（已离职）。课程可获取 Origin、ChemDraw、Photoshop 等化学专业常用软件学习版，并讲解其在化学化工中的应用。

给分不算高，期末考试较难，22 级为线上考试且题目更难，考试成绩直接计入课程成绩，可考虑选高年级跨专业选修以减轻压力。资料含杨甫老师各章 PPT 及课程作业图。

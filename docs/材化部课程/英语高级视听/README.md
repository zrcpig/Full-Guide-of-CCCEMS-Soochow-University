# 英语高级视听

> 特化型英语听力课程，旨在辅助六级听力备考，陈琳老师授课活泼。

## 学习建议

- 陈琳老师授课活泼，课堂以播放听力、完成课本听力题并讲解为主，常用雨课堂进行听力填空练习。
- 英语听力提升靠日常积累，每天多听 TED 比单纯上课更有效。
- 期中与期末考试大部分为课内听力，想考高分建议把书后听力稿通读记一遍，考场上可直接作答。
- 最后一题为听力命题大作文，类似高考作文，按需展开即可得分。

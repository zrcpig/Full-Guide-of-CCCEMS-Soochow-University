# 英语高级视听

> 本文件为该课程文件夹的资料导航。完整学习建议见 [英语高级视听.md](英语高级视听.md)。

## 内容介绍

英语高级视听是特化型英语听力课程，旨在辅助六级听力。陈琳老师授课活泼，以播放听力、完成课本题目并讲解为主，课堂常用雨课堂进行听力填空练习。

期中与期末考试大部分为课内听力，建议通读教材书后听力稿以便考场快速作答；最后一题为听力命题大作文，类似高考作文，按需展开即可得分。资料含听力稿及期末考试操作方案。

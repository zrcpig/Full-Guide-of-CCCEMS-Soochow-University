# 英语影视欣赏

> 特化型英语阅读课，服务于六级备考，何芊蔚老师授课口碑极佳。

## 学习建议

- 何芊蔚老师授课轻松有趣，课后作业少，学期中会反复提醒考核标准。
- 考核主要围绕课堂所教的几部英文电影展开，期中与期末各有约 50 分来自课内，建议完整观看课堂影片。
- MOOC 上的选择题多做做，题库答案已收入本文件夹。
- 第一个大作业：选择全英文视频片段自己配音，难度不高，可用 Pr 等工具制作。
- 第二个大作业：看电影做小组汇报。

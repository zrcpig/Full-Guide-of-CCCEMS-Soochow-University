# 英语影视欣赏

> 本文件为该课程文件夹的资料导航。完整学习建议见 [英语影视欣赏.md](英语影视欣赏.md)。

## 内容介绍

英语影视欣赏是特化型英语阅读课，同样服务于六级备考。何芊蔚老师的该课口碑极佳，课堂轻松有趣、课后作业少，学期中会反复提醒考核标准。

考核围绕课堂所教几部英文电影展开，期中与期末各有约 50 分来自课内，建议完整观看课堂影片并多做 MOOC 选择题。另有配音视频与小组汇报两项大作业，难度不高。资料含 MOOC 题库答案 PDF。

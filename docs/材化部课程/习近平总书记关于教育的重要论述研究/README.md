# 习近平总书记关于教育的重要论述研究

> 围绕习近平总书记关于教育的重要论述展开学习与研讨的思想政治理论/通识性质课程。

## 学习建议

"习近平总书记关于教育的重要论述研究"是一门面向本科生的公共必修或专题思政课程。课程内容涉及新时代教育方针、立德树人根本任务、教育强国建设等主题，通常以听课、雨课堂答题与小组/结课作业相结合的方式考核。

现收录的雨课堂答案文档可用于平时线上答题练习与复习，建议结合课堂进度及时完成雨课堂习题以保证平时成绩。

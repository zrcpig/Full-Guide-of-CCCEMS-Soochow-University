# 军事理论

> 面向全体本科生的国防教育类公共必修课，一般安排在大一开设。

## 学习建议

- 简称"军理"，由法学院过来的田毓老师讲授，被"流放"到材化来讲军理。
- 上课很有意思，田老师除考勤外几乎不会有任何要求。
- 课程涵盖国防基础、军事思想、国际战略环境、军事高技术等方面。
- **期末是思政类课程中最难的**，但田毓老师会给出一些提示，帮助同学顺利通过。
- 建议重点利用本文件夹内的思考题与一二三章题目进行针对性复习。

  - `苏州大学《军事理论》思考题.pdf`：思考题汇总，期末主观题答题素材。

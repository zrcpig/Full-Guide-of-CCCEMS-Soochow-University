# 军事理论

> 本文件为该课程文件夹的资料导航。完整学习建议见 [军事理论.md](军事理论.md)。

## 内容介绍

军事理论（简称"军理"）是面向全体本科生的公共必修课，属于国防教育类课程，一般安排在大一开设。课程内容涵盖国防基础、军事思想、国际战略环境、军事高技术等方面，旨在增强学生的国防观念与国家安全意识。

授课方面，主文档记录该课由法学院过来任教的田毓老师讲授。田老师讲课很有意思，对学生除考勤外几乎没有任何要求。期末考试是思政类课程中最难的，但田老师会给出一些提示，帮助同学顺利通过。备考时建议重点利用本文件夹内的思考题与一二三章题目进行针对性复习。

# 改革开放史

> "四史"课程之一，面向全校本科生的公共基础/思政必修课。

## 学习建议

- 22 级由佘明薇老师（现任马克思主义学院思政系主任）授课，老师十分体谅学生学习压力，讲课较为生动。
- **大作业**：以改革开放为主题，自选角度撰写个人理解，多写给分高。
- **期末考试**：全部为主观题，踩点给分，建议背诵指南中的思考题以便考场论述时言之有物。
- 编者曾任该课助教，目测给分不会低。

# 改革开放史

> 本文件为该课程文件夹的资料导航。完整学习建议见 [改革开放史.md](改革开放史.md)。

## 内容介绍

改革开放史是"四史"课程之一，属面向全校本科生的公共基础/思政必修课。22 级由佘明薇老师（现任马克思主义学院思政系主任）授课，老师十分体谅学生学习压力，讲课较为生动。编者曾任该课助教，目测给分不会低。

考核含大作业与期末两部分：大作业以改革开放为主题，自选角度撰写个人理解，多写给分高；期末考试全部为主观题，踩点给分，建议背诵指南中的思考题，以便考场论述时言之有物。本文件夹收录了大作业要求图与期末思考题 Word 文档，便于备考与完成作业。

# 化学专业英语

> 本文件为该课程文件夹的资料导航。完整学习建议见 [化学专业英语.md](化学专业英语.md)。

## 内容介绍

化学专业英语是化学类专业的一门专业必修/选修课，旨在培养学生阅读化学英文文献、进行化学专业英语翻译与表达的能力。课程内容涵盖化学专业词汇、元素周期表、无机/分析/有机等分支的英文表述与文献阅读，在培养方案中作为化学专业学生的语言工具课。

主文档记录该课由李亚红老师（701 一楼东北侧实验室）讲授。李老师非常负责也相当严格，**给分不会很高**。她喜欢随机点名（约两周一次，常课前点一次、课间再点一次），课上常叫人翻译句子或上黑板写单词（切勿用电子手环等作弊）。平时成绩取决于被点名次数与回答正确次数。第一节课作业通常是翻译元素周期表，可直接抄 *Inorganic Chemistry*，画格子、写整齐即可。

期末考试是材化部最严格的考试之一：考前 10 分钟清场，研究生逐桌检查，抽号入场并搜身，书包统一管理。考试难度不高，**重点在前几节课（无机部分）**，题型包括单词翻译、句子翻译、段落中英互译。本文件夹收录的 *Inorganic Chemistry* 与马永祥《化学专业外语》全文翻译是重要复习参考。

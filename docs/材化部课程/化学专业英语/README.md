# 化学专业英语

> 化学类专业必修/选修课，培养阅读化学英文文献、进行专业英语翻译与表达的能力。

## 学习建议

- 由李亚红老师（701 一楼东北侧实验室）讲授，非常负责也相当严格，**给分不会很高**。
- 李老师喜欢随机点名，频率约两周一次，常在课前点一次、课间再点一次。
- 课上常叫人翻译句子或上黑板写单词（切勿用电子手环等作弊）。
- 平时成绩取决于被点名次数与回答正确次数。
- 第一节课作业通常是翻译元素周期表，可直接抄 *Inorganic Chemistry*，画格子、写整齐即可。
- **期末考试是材化部最严格的考试之一**：
  - 考前 10 分钟清场，研究生逐桌检查。
  - 抽号入场并搜身，书包统一管理。
  - 考试难度不高，**重点在前几节课（无机部分）**。
  - 题型：单词翻译、句子翻译、段落中英互译。
- 本文件夹收录的 *Inorganic Chemistry* 与马永祥《化学专业外语》全文翻译是重要复习参考。

## 资料说明

- `《Inorganic Chemistry》(10).pdf`：无机化学英文教材，第一节课翻译元素周期表作业的抄写来源，也是期末考试无机部分重点的对应参考书。
- `全文翻译-化学专业外语-马永祥-兰州大学.pdf`：马永祥《化学专业外语》（兰州大学）全文翻译，化学专业英语学习的经典参考，可用于翻译练习与词汇积累。

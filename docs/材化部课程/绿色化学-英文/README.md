# 绿色化学-英文

> 本文件为该课程文件夹的资料导航。完整学习建议见 [Green_Chemistry.md](Green_Chemistry.md)。

## 内容介绍

绿色化学（英文）是材化部的一门英文授课专业课程，由李新明教授主讲。课程围绕绿色化学的基本理念与技术展开，涵盖绿色化学定义、反应物、绿色催化、绿色溶剂、绿色工艺设计、可再生资源等主题。

课程考核包含期中英文 Presentation 与期末闭卷考试。资料以李新明老师课堂 PPT（class 1–7）和复习提纲为主，每个主题配有 pre-reading 阅读材料，便于课前预习与期末梳理。

# 分析化学实验

> 本文件为该课程文件夹的资料导航。完整学习建议见 [分析化学实验.md](分析化学实验.md)。

## 内容介绍

分析化学实验是化学类专业的专业基础实验课，分两学期开设：上学期为分析化学实验（化学分析实验），下学期为仪器分析实验。课程在培养方案中与分析化学/仪器分析理论课配套，是培养学生定量分析操作与仪器使用能力的核心实践环节。

主文档说明：上学期分析化学实验与无机化学实验类似，22 级可选老师有李晓虹、严吉林、李建国、龙玉梅、王杰、何学文、王炯等，各老师风格与给分松紧不同（如李建国最水、王炯推荐、龙玉梅对女生较照顾、何学文较轴但负责等）。下学期仪器分析实验统一在 705（材化部液氮塔旁）进行，轮流到各实验室做不同项目，由各老师负责，期末为**实验操作考试**。

本文件夹主要收录下学期仪器分析实验的资料，包括与各实验一一对应的实验报告（PDF）、实验原理与操作视频（mp4）、报告模板（docx），以及 19 级期末考试试卷，是复习操作要点与撰写报告的实用参考。

## 资料说明

- `分析化学实验.md`：课程主文档，说明上下学期安排与各授课老师特点。
- `仪器分析实验实验报告/`：各仪器分析实验的报告 PDF（ICP、分光光度法、原子吸收、台式核磁、氟离子选择电极、电位滴定、循环伏安、库仑滴定、离子色谱、荧光、红外、气相/液相色谱等约 20+ 个实验）。
  - `实验报告模板/`：15 个实验对应的 docx 报告模板，撰写报告时直接套用。
- `仪器分析实验实验视频/`：与上述实验一一对应的原理与操作视频（mp4），每个实验通常含"实验原理"和"实验操作"两段，便于预习与复习操作要点。
- `仪器分析实验教案/`：气相色谱相关教案视频（2 个 mp4）。
- `仪器分析实验期末/`：期末操作考试复习材料。
  - `19级仪器分析考试.pdf`：19 级仪器分析实验期末考试试卷。

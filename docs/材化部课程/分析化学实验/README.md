# 分析化学实验

> 化学类专业的专业基础实验课，分两学期开设：上学期为分析化学实验（化学分析实验），下学期为仪器分析实验。

## 学习建议

分析化学实验包括两部分，上学期的是分析化学实验，下学期的是仪器分析实验。

### 分析化学实验上

与无机化学实验类似，22 级可选教师有李晓虹、严吉林、李建国、龙玉梅、王杰、何学文、王炯：

- **李晓虹老师**：强化班限定，教课比较水。
- **严吉林老师**：强化班限定，比李稍严格。
- **李建国老师**：最水，上课你只要不爆炸他一般不会管。
- **龙玉梅老师**：女生选龙老师会得到额外照顾，男生选比较坐牢。
- **王杰老师**：较水，上课说话声音比较小，新老师。
- **何学文老师**：比较轴，上课很负责，给分可以。
- **王炯老师**：水，推荐。

### 分析化学实验下（仪器分析实验）

统一在 705（材化部液氮塔旁边的楼）做，轮流去每个实验室做，由各个老师负责，期末考试为实验操作考试。

本文件夹主要收录下学期仪器分析实验的资料，包括与各实验一一对应的实验报告（PDF）、实验原理与操作视频（mp4）、报告模板（docx），以及 19 级期末考试试卷，是复习操作要点与撰写报告的实用参考。

## 文件夹架构

```
分析化学实验/
├── 仪器分析实验实验报告/  …（约22个PDF + 实验报告模板子目录，含15个docx模板）
│   └── 实验报告模板/  …（15个实验的docx模板）
├── 仪器分析实验实验视频/  …（约28个mp4，每个实验含"原理""操作"两段）
├── 仪器分析实验教案/  …（气相色谱相关2个mp4）
└── 仪器分析实验期末/
    └── 19级仪器分析考试.pdf
```

## 资料说明

- `仪器分析实验实验报告/`：各仪器分析实验的报告 PDF（ICP、分光光度法、原子吸收、台式核磁、氟离子选择电极、电位滴定、循环伏安、库仑滴定、离子色谱、荧光、红外、气相/液相色谱等约 20+ 个实验）。
  - `实验报告模板/`：15 个实验对应的 docx 报告模板，撰写报告时直接套用。
- `仪器分析实验实验视频/`：与上述实验一一对应的原理与操作视频（mp4），每个实验通常含"实验原理"和"实验操作"两段，便于预习与复习操作要点。
- `仪器分析实验教案/`：气相色谱相关教案视频（2 个 mp4）。
- `仪器分析实验期末/`：期末操作考试复习材料。
  - `19级仪器分析考试.pdf`：19 级仪器分析实验期末考试试卷。

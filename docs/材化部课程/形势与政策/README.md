# 形势与政策

> 本文件为该课程文件夹的资料导航。完整学习建议见 [形势与政策.md](形势与政策.md)。

## 内容介绍

形势与政策（简称"形策"）是面向全校本科生的公共基础必修课，贯穿本科四个学年，由马克思主义学院组织，依托智慧树等平台开展线上修读。课程以观看视频、完成作业与期末课程论文为主要考核形式。

学习建议上，严格按照学校下发的修读通知与课程要求走就不会漏环节；视频刷完后不要急着做题，网上一般能找到最新全对的答案可直接参考。期末需按题目与格式要求撰写课程论文。本文件夹收录了 2022 级的修读通知、课程要求、期末论文题目及格式要求，以及智慧树修读作业说明截图，供对照流程使用。

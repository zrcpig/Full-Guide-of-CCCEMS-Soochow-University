# 形势与政策

> 面向全校本科生的公共基础必修课（简称"形策"），贯穿本科四个学年，依托智慧树等平台开展线上修读。

## 学习建议

- 严格按照学校下发的修读通知与课程要求走就不会漏环节。
- 刷完视频别急着做题，网上一般有最新的全对答案，直接参考。
- 期末需按题目与格式要求撰写课程论文。
- 本文件夹收录了 2022 级的修读通知、课程要求、期末论文题目及格式要求等资料，供对照流程使用。

# 教育技术与教育科研方法

> 面向师范方向学生的专业基础课，融合教育技术应用与教育科研方法论两大模块。

## 学习建议

教育技术与教育科研方法通常在本科中段修读。课程融合教育技术应用与教育科研方法论两大模块，内容涵盖教学媒体与信息化教学设计、教育研究的基本流程与常用方法（如文献研究、调查研究、实验研究、个案研究等），旨在培养师范生开展教育科研与信息化教学的能力。

本文件夹提供课程 Word 资料，可作为复习与作业参考。

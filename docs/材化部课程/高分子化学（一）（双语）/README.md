# 高分子化学（一）（双语）

> 材化部专业核心课，22 级可选张正彪或朱健老师两班。

## 学习建议

- 22 级可选两位老师：张正彪老师本人较少直接授课；朱健老师班由朱健与宣孙婷两位老师共同授课，不布置作业但对自学能力要求较高。
- 期中考试重点看张正彪老师的习题课 PPT，能覆盖约 80% 的考点，剩余零碎知识点需通读其 PPT。
- 期中题型包括中英互译名词解释（用简短话语解释名词，占约 60%）与计算题（如凝胶点计算）。
- 计算题公式可背诵整理资料（期中复习里的朱健老师 PDF）。

# 健康标准测试

> 面向全体在校本科生的体质健康测试项目，成绩计入综合素质评价与评奖评优。

## 学习建议

- 简称"体测"，一般大三大四每年进行一次。
- 测试项目包括身高体重、肺活量、50 米跑、坐位体前屈、立定跳远、引体向上（男）/仰卧起坐（女）、1000 米（男）/800 米（女）跑等。
- 2025 年以后体测可能由人工测试改为机器测试。
- 2025 年起苏州大学体测及格分改为 60 分。
- 推荐使用微信小程序"大学生体测分数计算器"提前估算各项目得分，有针对性地练习薄弱项目，确保达到及格线以上。
- 相关体测题库与须知见 `公共体育` 文件夹。

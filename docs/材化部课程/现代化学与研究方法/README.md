# 现代化学与研究方法

> 现代化学与研究方法是材化部的专业选修课，总体属"水课但给分不高"。

## 学习建议

- 这门课由三位老师共同授课。首先是[屠一峰](https://chemistry.suda.edu.cn/31/75/c31983a405877/page.htm)老师，讲课非常随意，喜欢写书法，属于学部的养老型教授，每天喝茶搞艺术那种。
- 然后是学部副主任姚英明老师，上过文献检索的都知道这个老师比较水，但在现代化学这门课上他最喜欢点名，经常在课间搞突击点名。
- 最后是[屠迎峰](https://chemistry.suda.edu.cn/31/05/c31985a405765/page.htm)，毕业于北京大学，上课特别喜欢讲北大那些事，也是三人里最喜欢随机点名回答问题的。
- 课程最后先安排小组汇报，主题不限，建议直接照搬别的课做过的汇报，不要浪费时间。
- 最后一节课由屠一峰老师用高超的板书技巧在黑板上写下小论文的主题，800 字左右即可，写的越多相对给分越高。
- 总体来说属于专业选修里的**水课，但给分不高**。

# 现代化学与研究方法

> 本文件为该课程文件夹的资料导航。完整学习建议见 [现代化学与研究方法.md](现代化学与研究方法.md)。

## 内容介绍

现代化学与研究方法是材化部的专业选修课，由三位老师共同授课：首先是[屠一峰](https://chemistry.suda.edu.cn/31/75/c31983a405877/page.htm)老师，讲课随意、喜欢写书法，属学部养老型教授；其次是学部副主任姚英明老师，最喜欢课间突击点名；最后是毕业于北京大学的[屠迎峰](https://chemistry.suda.edu.cn/31/05/c31985a405765/page.htm)老师，喜欢讲北大往事，也是三人中最爱随机点名回答问题的。

考核方式为先做小组汇报（主题不限，建议直接照搬其他课做过的汇报），再在最后一节课由屠一峰老师板书小论文主题（800 字左右，写多给分相对更高）。总体属专业选修里的水课，但给分不高。

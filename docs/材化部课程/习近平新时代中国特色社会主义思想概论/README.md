# 习近平新时代中国特色社会主义思想概论

> 高校思想政治理论课核心课程，系统讲授"十个明确""十四个坚持"等核心要义。

## 学习建议

- 简称"习概"，面向全体本科生的公共必修课，一般在本科中段开设。
- 大作业为小组汇报。
- 期末考试全部为主观题，**开卷考试**。
- 复习策略：把仓库里的资源打印下来，考前半小时看一下，直接考场上参考作答即可。
- 掌握好本文件夹内的课后习题答案与重点知识点总结是应对开卷考试的关键。

## 文件夹架构

```
习近平新时代中国特色社会主义思想概论/
└── 习概课程资料/
    ├── 习概2023版课后习题参考答案.pdf
    ├── 习概重点知识点总结.pdf
    └── 习概重点知识点总结2.pdf
```

## 资料说明

- `习概课程资料/`：开卷考试备考资料目录。
  - `习概2023版课后习题参考答案.pdf`：2023 版教材课后习题参考答案，开卷考试可直接参考。
  - `习概重点知识点总结.pdf`、`习概重点知识点总结2.pdf`：重点知识点汇总，建议打印带入考场。

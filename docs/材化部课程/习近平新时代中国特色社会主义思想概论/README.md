# 习近平新时代中国特色社会主义思想概论

> 本文件为该课程文件夹的资料导航。完整学习建议见 [习近平新时代中国特色社会主义思想概论.md](习近平新时代中国特色社会主义思想概论.md)。

## 内容介绍

习近平新时代中国特色社会主义思想概论（简称"习概"）是高校思想政治理论课的核心课程之一，属于面向全体本科生的公共必修课，一般在本科中段开设。课程系统讲授习近平新时代中国特色社会主义思想的核心要义、精神实质与丰富内涵，涵盖"十个明确""十四个坚持"等内容。

考核方面，主文档指出该课大作业为小组汇报，期末考试全部为主观题且为**开卷考试**。复习策略上，把本仓库收录的资料打印下来、考前半小时浏览一遍，即可在考场上直接参考作答。因此，掌握好本文件夹内的课后习题答案与重点知识点总结是应对开卷考试的关键。

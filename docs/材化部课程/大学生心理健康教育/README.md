# 大学生心理健康教育

> 面向全校本科生的公共基础/通识必修课（简称"大心"），以心理健康常识普及与心理调适方法为主。

## 学习建议

- 一门水课，学业压力较小。
- 22 级由王静老师授课，人品很好，上课经常让大家开怀大笑。
- 期末考核为随堂撰写一篇大作文，主题由老师自定，只要认真完成写作，给分一般不低于 90 分。
- 只需按时到课、完成期末作文即可获得较高分数，无需额外复习。

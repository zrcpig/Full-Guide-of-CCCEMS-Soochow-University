# 化工基础

> 化学班、化工班、应用化学班共同的专业必修课，涵盖流体流动与输送、热量传递、传质分离等化工原理内容。

## 学习建议

- 由于材化部中化工学院非常小，授课老师目前为伍鹏（厦门大学生物工程系）。
- 课程后半部分的化学反应工程是材化部选修课（化工专业必修），教材基本上只学一半。
- 课程比较考验自学能力，作为早八课程课上能听懂的内容有限，考试考的都是 PPT 上例题的变种，**务必把 PPT 完完整整啃下来**。
- **期中考试**：期中前学完第一章流体力学，注意期中前最后一节课讲的泵相关知识；期中尤其考察选择题与判断题，计算题难度不高，重点在简答题和前面的选择判断。
- **期末考试**：选择判断题变简单，计算题占比增加，反应塔不会考很难，把 PPT 上的例题全部做一遍即可。

## 资料说明

- `伍鹏老师的PPT/`：伍鹏老师授课课件与配套材料，考试例题的主要来源，必须完整掌握。
  - 绪论及第一/二/三章 PDF：各章节课件，对应流体流动及输送、热量传递、传质分离等内容。
  - `化工基础 计算题（题）- 课程中心.pdf`：课程中心计算题集，计算题复习核心。
  - `化工基础作业1/2/3.pdf`：三次作业，配合课件练习。
  - `化工基础考试.m4a`：考试相关录音，可用于了解考查重点。
- `化工基础 期中/`：期中考试资料。
  - `苏州大学2022级化工基础期中考试.pdf`：2022 级期中试卷，期中复习与摸底用。

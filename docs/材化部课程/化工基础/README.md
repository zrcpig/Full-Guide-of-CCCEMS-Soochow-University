# 化工基础

> 本文件为该课程文件夹的资料导航。完整学习建议见 [化工基础.md](化工基础.md)。

## 内容介绍

化工基础是化学班、化工班、应用化学班（22 级专业）共同的专业必修课，属于化工方向的大类基础/专业核心课程。课程涵盖流体流动与输送、热量传递、传质分离等化工原理内容，后半部分的化学反应工程也是材化部的一门选修课（化工专业必修），因此本课教材基本上只学一半。

授课方面，主文档指出由于材化部化工学院规模较小，目前能上这门课的就是伍鹏老师（厦门大学生物工程系）。该课为早八课程，课上能听懂的内容有限，比较考验自学能力。考试考的都是 PPT 上例题的变种，因此**务必把 PPT 完完整整啃下来**。

复习要点：期中前学完第一章流体力学，注意期中前最后一节课讲的泵相关知识，期中考试尤其考察选择题与判断题，计算题难度不高，重点在简答题与前面的选择判断；期末考试选择判断变简单、计算题占比增加，反应塔不会考很难，把 PPT 例题全部做一遍即可。本文件夹收录的伍鹏老师 PPT、作业、计算题与期中试卷正是按此策略复习的核心材料。

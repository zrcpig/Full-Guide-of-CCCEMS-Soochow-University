# 分析化学

> 本文件为该课程文件夹的资料导航。完整学习建议见 [分析化学.md](分析化学.md)。

## 内容介绍

分析化学是化学类专业的专业基础课（大类基础/专业必修），分两学期开设：上学期为分析化学（化学分析），下学期为仪器分析。课程在培养方案中是化学、应用化学、化工等专业的核心课程，也是后续分析化学实验、仪器分析实验的理论基础。

主文档指出：上学期苏大使用武汉大学的分析化学教材，并配有习题解答，建议收一本二手。武大教材"防自学系统"较强，自学困难，上课听不懂也正常，建议**抛开武大教材**，直接看原版外文 *Daniel C. Harris, Quantitative Chemical Analysis, 8th Edition*（武大教材借鉴自该书）。上学期考试需多刷武大配套习题解答，例题尤为重要。

下学期使用方慧群老师的仪器分析教材，但教材质量一般且国内无更好替代，建议**专攻 PPT**。仪器分析考试直接背苏州大学题库，会考原题，背下题库可稳拿 80 分以上。本文件夹按上/下学期分别整理了各老师 PPT、小测、期中/期末试卷与资料，便于针对性复习。

## 资料说明

- `分析化学.md`：课程主文档，说明上下学期教材选择、自学建议与题库复习策略。
- `分析化学上/`、`分析化学下/`：按学期组织的全部资料。
  - `Daniel-C.-Harris-...pdf`：原版外文教材，上学期自学主力，建议以此替代武大教材。
  - `分析化学 习题解答 上册 (曾百肇...).pdf`：武大配套习题解答，上学期考试刷题核心，例题重要。
  - `周群老师的PPT/`、`郭文英老师的PPT/`、`王杰老师的PPT/`：各位授课老师的课件，下学期尤其以 PPT 为主。
  - `分析化学上小测/`、`分析化学下小测/`：历届小测题，按年级整理，用于平时练习与摸底。
  - `分析化学上期中/`、`分析化学下期中/`：历届期中试卷（下学期含 14—22 级及讲评），重点复习材料。
  - `分析化学下期末/`：历届期末试卷（15—20 级）。
  - `分析化学上资料/`、`分析化学下资料/`：计算题、题库、练习卷、复习提纲及方惠群《仪器分析》课后答案等；**下学期题库为考试原题来源，务必背熟**。

# 分析化学

> 化学类专业的专业基础课（大类基础/专业必修），分两学期开设：上学期为分析化学（化学分析），下学期为仪器分析。

## 学习建议

### 分析化学上（化学分析）

苏大使用的教材是武汉大学的《分析化学》，这本书有一本配套的习题解答，建议收一本二手。武大写的分析化学突出一个不当人，有极强的防自学系统，自学基本看不懂，上课听不懂也很正常。**建议抛开武大教材**，直接去看原版外文的 *Daniel C. Harris, Quantitative Chemical Analysis, 8th Edition*（武大的教材是"借鉴"了这本教材）。这本书请在库里找。

分析化学上的考试需要多刷武大配套的分析化学习题解答，里面的例题很重要，需要多刷。

### 分析化学下（仪器分析）

苏大使用的教材是方慧群老师的《仪器分析》，写得一般，但国内没有什么更好的替代，建议把书扔掉专攻 PPT 即可。

分析化学下（仪器分析）的考试直接背苏州大学题库，里面会考原题，把题库背下来能拿 80 分以上没问题。

## 资料说明

- `Daniel-C.-Harris-...pdf`：原版外文教材，上学期自学主力，建议以此替代武大教材。
- `分析化学 习题解答 上册 (曾百肇...).pdf`：武大配套习题解答，上学期考试刷题核心，例题重要。
- `周群老师的PPT/`、`郭文英老师的PPT/`、`王杰老师的PPT/`：各位授课老师的课件，下学期尤其以 PPT 为主。
- `分析化学上小测/`、`分析化学下小测/`：历届小测题，按年级整理，用于平时练习与摸底。
- `分析化学上期中/`、`分析化学下期中/`：历届期中试卷（下学期含 14—22 级及讲评），重点复习材料。
- `分析化学下期末/`：历届期末试卷（15—20 级）。
- `分析化学上资料/`、`分析化学下资料/`：计算题、题库、练习卷、复习提纲及方惠群《仪器分析》课后答案等；**下学期题库为考试原题来源，务必背熟**。

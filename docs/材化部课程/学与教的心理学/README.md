## 内容介绍

学与教的心理学（教育心理学）是面向师范方向学生的师范方向必修/专业基础课，通常在本科中段修读，是教师资格认定与教师专业成长的核心理论课程。课程内容涵盖学习理论、教学心理、学生心理发展与个体差异等模块。

本文件夹暂无独立学习建议文档（无 `.md` 文件），仅提供《教育心理学期末复习知识点》Word 文档，可用于期末梳理知识框架、突击背诵核心考点。

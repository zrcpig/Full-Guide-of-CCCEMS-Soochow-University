# 学与教的心理学

> 面向师范方向学生的专业基础课，是教师资格认定与教师专业成长的核心理论课程。

## 学习建议

学与教的心理学（教育心理学）通常在本科中段修读。课程内容涵盖学习理论、教学心理、学生心理发展与个体差异等模块。

本文件夹提供《教育心理学期末复习知识点》Word 文档，可用于期末梳理知识框架、突击背诵核心考点。

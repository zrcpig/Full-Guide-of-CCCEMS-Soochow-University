# 教务系统选课

> 选课时间、选课规则、退选注意事项等

选课时间：一半是在上一学期的期中考试附近选定下一学期的课

选课规则：拼手速

- 教务系统会在12点15左右开放选课，所以需要提前规划好自己到底要选什么课。
- 原则：能选水课不选雷课；能选简单课不选难课；别给自己找不痛快（某些老师的课学得最认真反而给分低，选错会极大打击你的学习兴趣）
- 焚决：12点那批没抢上心仪的课不用慌，一般会有几天的调换补选时间，重点关注凌晨12点、下午3点、晚上8点等几个关键时间段，会有卖课换课的同学，此时捡漏截胡概率高。

退选注意事项：

- 退选课前一定要对照**培养方案**算够学分（在材化部官网找）
- 每学期开学的第一周是体验课阶段，这个时候感觉某门课较雷，别将就自己，该退退，该换换
- 退课尽量不要退必修课，否则错过开课时间补起来很麻烦
- 材化部有免修政策，要求你所有课程绩点3.6以上即可申请，教务会请老师给你出一份免修课程的较难的考试题，你考多少分这门课就是多少分（80以下免修不通过，重新选课上课）（建议有条件就试试，不亏）

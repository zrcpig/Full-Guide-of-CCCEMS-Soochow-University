# 志愿活动换志愿时长

> 志愿时长与毕业没有任何关系，不要听信谣言；仅评奖评优对志愿时长有限制，每学年能做够60个小时就满足95%的评奖评优要求了

材化部青协有专门的QQ群每天发布最新的志愿活动，各类志愿活动性价比不同：
1. 志愿时长：真实时长 = 1：1
    - 纯苦力活的搬书；搬东西
    - 往天赐庄送文件
    - 小手工制作等
    - 红十字会组织的相关志愿（苦、累）
2. 志愿时长：真实时长 = （1~2）：1
    - 材化部团委指派的活动（如蔡X娣博物馆值班；金鸡湖拾垃圾等）（考核较水，基本不花多长时间）
    - 苏州博物馆、苏州火车站、苏州大学图书馆、独墅湖图书馆值班（考核较水，可以有很长时间干自己事情）
    - 各类寒暑假志愿者（实在闲的没事干可以去，对自己提升不大）
    - 寒假回访母校活动，夯，既可以回去母校公开演讲锻炼表达能力，还可以名正言顺得苏大志愿时长。
3. 志愿时长：真实时长 = （3~4）：1
    - 菜花小灶（跃迁课堂、材化小讲师），通常是给低一级的同学周末讲讲课，负责任的话需要自备讲义等，对个人能力有要求，但性价比很高
    - 寒暑假支教团，通常是自费车票（材化管食宿），既可以旅游长见识（但住宿往往是在乡下，做好防蚊驱虫准备），又可以美美赚时长，如果登上公众号报道的话往往能干到100+小时，性价比高
    - 学校共青团委下属学生组织，如共青团校、学生会等，每年都会结算一次志愿时长

此外，学校还提供了校内跨学院志愿时长兑换和校外志愿时长兑换，但是根据材化部政策，这两种途径获得的志愿时长均有上限（50h，以材化青协当年通知为准），**总体不推荐去别的学院搞志愿**。

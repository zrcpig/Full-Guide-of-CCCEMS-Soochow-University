# 材化部推荐入党积极分子

入党的全流程如下：
1. 递交入党申请书
    - 年满十八周岁，直接交给辅导员即可
    - 日期尽量往前写（可以直接写成开学时间）
2. 党组织派人谈话（走形式）
3. 推荐和确定入党积极分子
    - 由所在班级团支部推优（名额为班级团员人数的1/10）
    - 由所在学生组织推优（名额较少，一般3~4个）
    - 建议在大一学年推优成功，到了大二大三学年后排名竞争压力大，很难给名额
4. 上级党委备案（走形式）
5. 指定培养联系人（走形式）
6. 培养教育考察
    - 材化部党校会上党课给入党积极分子
    - 会发一张材化部党校盖章的结业证书
7. 确定发展对象（**最难**）
    - 材化部按当年入党积极分子的约1/10给名额
    - 按所有专业总绩点排名进行排序，从前到后给定名额（决定了强化班天生名额多）
    - 普通专业（如普班化学）几乎每年只能分到1~2个（甚至0）
8. 报上级党委备案（走形式）
9. 确定入党介绍人（走形式）
10. 进行政治审查
    - 会往父母工作单位寄政审表（尽量找好盖章的单位盖）
11. 开展集中培训
    - 上党课，写思想汇报（不想写可看我Github库）（党课也有速刷脚本）
12. 支委会审查（走形式）
13. 上级党委预审（走形式）
14. 填写《入党志愿书》
    - 需要约1天完整的时间进行誊写，要写的东西非常多
15. 支部大会讨论
    - 人员已经内定，纯走过场
16. 上级党委派人谈话（走形式）
17. 上级党委审批（走形式）
18. 上级党委组织备案（走形式）
19. 编入党支部和党小组（走形式）
20. 入党宣誓（走形式）
21. 继续教育考察
    - **别挂科！别挂科！别挂科！**
22. 提出转正申请（走形式）
23. 支部大会讨论
    - 已内定，纯走过场
24. 上级党委审批（走形式）
25. 材料归档
    - 正式成为党员

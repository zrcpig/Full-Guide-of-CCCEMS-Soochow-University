# 教师资格证

> 报考条件、报名时间、备考资料等

报考时间：

- 一般上半年一次下半年一次，上半年在3月份报名；下半年在9月份报名
- 笔试与报名时间相差一个月左右，面试与笔试时间相差一个月左右

报考条件：

- 一般要求在校的大三、大四年级方能报考
- 大三通过考试（笔试、面试）后不能立即拿证，只有在大四下半学期才能提交申请去苏州市政务服务中心拿证
- 不限制师范专业报考，应考尽考

备考方式：

- **不需要**报班，**不需要**报网课，你唯一需要就是收一套科目一二三的教材（和往年真题）
- 笔试提前一个月看书背诵（极限背诵也需要1个星期的时间，不要把自己逼太紧）
- 笔试整体体验和考文科综合无异，主要就是论述题、判断题、选择题等，科目一会让你写篇作文；科目二会有大量的类似高中政治的题目（按高中结构化答题即可踩点得分）；科目三基本上只要能考上苏州大学的，这点水平的题目应该没问题（重点练一下写教案）
- 面试**不需要**买书，网上有大量整理现成的往年教案，练习10~20个即可，每年的题库都会换，重点是学方法，不要死记硬背
- 自己叫上同学在台下听听讲课并给你指出缺点，不断改进即可
- 面试的教案不做为评分的参考，你心中有数就行，考场上教案只给你自己看，不需要写得多精美

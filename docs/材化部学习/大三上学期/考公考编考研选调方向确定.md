# 考公考编考研选调方向确定

> 各类升学就业路径的时间规划、准备建议等

材化部升学整体情况：

- 一般而言，材化部本科生升学率会在50~60%之间（含保研、考研、出国）
- 保研去向详见保研章节（留在本校的学生一般在保研总人数的50%）
- 考研去向（约30%的人考的还是本校，剩下的一般会往985考（吉大、川大、湖大、中科大、南大等））
- 出国去向（南洋理工大学最多，新加坡国立大学次之，偶有蒙纳什/帝国理工/新南威尔士等）

升学时间规划：

- 最好在大二下之前就有明确规划，保研规划最迟不要超过大二下，否则很多东西来不及补
- 考研的话大三上要开始准备，大三升大四的12月考试，因此从1月份开始你就要准备，备考一整年（需要具体学校建议的可以积极咨询学长学姐）
- 出国的话需要提前考掉雅思/TOFEL，雅思在苏大天赐庄北区有考点（据我同学说苏大的口语考官比西交利物浦的给分高）

材化部就业整体情况：

- 材化部材料科学与工程专业为双一流专业，可参加各省选调
- 苏州大学在江苏省内用人单位中较为欢迎，考公考编有优势

就业时间规划：

- 如果家里有资源的（江苏省内学生），听你家里的安排，家里人不会给你安排差的
- 如果家里没有资源的，海投材料、涂料、生物医药企业，任何岗给钱就投；材化本科最高工资一般出现在教培行业
- 本科积极转行的，根据你的专业能力合理规划
- 苏州大学线下摆摊的校招都是垃圾企业（不严谨）/线下摆摊的一般不会给好岗位，真正的好企业好岗位需要自己去企业官网海投（别信什么BOSS直聘，91job等）

# 联系老师进课题组

> 如何联系老师、入组前的准备、实验室选择建议等

如何联系老师：

- 一般你可以在材化部官网上直接找到各个老师的个人介绍和代表文章
- 如果想了解老师的最新动态，可以Google或者通过ORCid等查询
- 如果确定了老师，建议使用学生邮箱（stu.suda.edu.cn）给老师发邮件，避免被反垃圾规则屏蔽
- 给老师发邮件的内容一般包括：
    - 自己的个人信息
    - 学习绩点及排名
    - 科研意向和进组目标
- 选老师的原则：
    - 首先要人品端正，爱护学生，乐教善教
    - 其次要选一些研究方向比较新颖，仪器设备比较全面的老师（有起码的学术能力）
    - 最后带本科生课程的老师一般脾气会好一点（有例外，别杠）

入组前的准备：

- 本校老师大部分不需要你在进组前有任何准备，老师会安排研究生对你进行指导
- 如果进组目的是为了打竞赛/发文章，**一定**要和老师提前说明，给你分配相应资源，否则老师只会给你最低程度的指导

实验室选择建议：

- 705的5楼ICCS
- 912材料大楼的高楼层（高分子国地实验室）
- 纳米科学与技术学院
- 其它挂牌子有title的实验室（包括不限于：中葡一带一路实验室；苏州大学水资源处理重点实验室等），这种实验室一般会有较好的设备

# 英语六级（CET-6）

> 历年英语六级真题与听力音频，按考试时间分目录。

## 内容介绍

收录 2018.06 至 2023.06 共 14 次六级考试的真题、详解与听力音频，按 `年.月` 分目录。每次考试通常含 3 套真题、对应详解及听力 mp3。

## 资料说明

每个考试目录通常包含：

| 文件类型 | 用途 |
| :--- | :--- |
| `*.英语六级真题第N套.pdf` | 第 N 套真题试卷 |
| `*.六级真题第N套详解.pdf` | 第 N 套真题答案与详解 |
| `*.六级真题第N套听力音频.mp3` | 第 N 套听力音频 |

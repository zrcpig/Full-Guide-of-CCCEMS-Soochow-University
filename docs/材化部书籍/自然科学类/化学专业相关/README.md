# 化学专业相关书籍

> 化学各分支的经典教材与参考书（中英文），含分析、化工、无机、有机、物化、计算化学、量子化学。

## 内容介绍

按化学分支分目录整理，每个目录内为该分支的中英文经典教材。可配合对应课程文件夹使用，加深对课内知识的理解。

## 文件夹架构

```
化学专业相关/
├── 分析化学/      # 武大分析化学、Harris、Skoog、Christian 等中外教材
├── 化工原理/      # 化学工程基础教材
├── 无机化学/      # 《Inorganic Chemistry》、格林伍德《元素化学》、徐如人《无机合成与制备化学》
├── 有机化学/      # McMurry、Clayden 等有机经典教材及题解
├── 物理化学/      # 傅献彩物化、Atkins 物化（中英文）、分子 approaches
├── 计算化学/      # G09 中文说明书等
└── 量子化学/      # Szabo、Levine、格里菲斯、福井谦一、钱伯初等量子力学/量子化学教材
```

## 资料说明

| 目录 | 用途 |
| :--- | :--- |
| `分析化学/` | 分析化学中外经典教材，配套无机/分析化学课程 |
| `化工原理/` | 化学工程基础参考书，配套化工基础课程 |
| `无机化学/` | 无机化学与元素化学经典教材，配套无机化学课程 |
| `有机化学/` | 有机化学中外经典教材及题解，配套有机化学课程 |
| `物理化学/` | 物理化学中外经典教材，配套物理化学课程 |
| `计算化学/` | 计算化学入门与 Gaussian 使用说明 |
| `量子化学/` | 量子力学与量子化学教材，配套量子化学基础、结构化学课程 |

# 高数专业相关书籍

> 高等数学与数学分析相关教材及习题集。

## 内容介绍

含苏大高数课本、吉米多维奇习题精选、程艺《数学分析讲义》及同步练习。可供修读高等数学、数学分析时配合刷题使用。


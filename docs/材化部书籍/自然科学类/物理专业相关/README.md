# 物理专业相关书籍

> 物理相关经典教材，含大学基础物理、朗道理论物理教程、量子力学。

## 内容介绍

按物理分支分目录整理。其中朗道理论物理教程为理论物理经典全套，量子力学部分含 Cohen-Tannoudji、文小刚多体理论等。可供修读普通物理、量子化学基础等课程时拓展阅读。

## 文件夹架构

```
物理专业相关/
├── 大学基础物理/    # 基础物理学（上/下）电子教材
├── 朗道理论物理/    # 朗道理论物理教程（卷01-09）中译本
└── 量子力学/        # Cohen-Tannoudji、文小刚多体理论、相对论等
```

## 资料说明

| 目录 | 用途 |
| :--- | :--- |
| `大学基础物理/` | 基础物理学电子教材，配套普通物理课程 |
| `朗道理论物理/` | 朗道-栗弗席兹理论物理教程，力学/场论/量子力学/统计物理等 |
| `量子力学/` | 量子力学经典教材，含多体理论与相对论 |

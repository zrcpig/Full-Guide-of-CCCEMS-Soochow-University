# 材化部选课

> 本文件夹汇总苏州大学材化部选课相关的培养方案、推荐课表与学分认定文件。

## 内容介绍

含材化部 22 级本科培养方案与各学期推荐课表，以及创新创业学分认定条例，便于规划选课与学分。

## 文件夹架构

```
材化部选课/
├── 22本科培养方案.pdf                          # 材化部 2022 级本科培养方案
├── 2022级推荐课表-2022-2023-2.pdf              # 大一下学期推荐课表
├── 2022级推荐课表-2023-2024-1.pdf              # 大二上学期推荐课表
├── 2022级推荐课表-2023-2024-2.pdf              # 大二下学期推荐课表
├── 2022级推荐课表-2024-2025-1.pdf              # 大三上学期推荐课表
├── 2022级推荐课表-2024-2025-2.pdf              # 大三下学期推荐课表
└── 2022年修订-创新创业学分认定管理条例.pdf      # 创新创业学分认定办法
```

## 资料说明

| 文件 | 用途 |
| :--- | :--- |
| `22本科培养方案.pdf` | 材化部 2022 级培养方案，含毕业学分要求与课程模块 |
| `2022级推荐课表-*.pdf` | 各学期推荐选课表，按学期给出建议课程组合 |
| `2022年修订-创新创业学分认定管理条例.pdf` | 创新创业学分认定标准与流程 |

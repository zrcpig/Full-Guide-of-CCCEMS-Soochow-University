# 材化部推荐免试研究生

> 本文件夹汇总苏州大学材化部推荐免试研究生（保研）相关资料。

## 内容介绍

本文件夹含苏大材化推免去向汇总及建议，以及推免相关的官方通知与申请文件，供有志保研的同学参考。

## 文件夹架构

```
材化部推荐免试研究生/
├── 苏大材化推免去向汇总及建议.md   # 历年推免去向分析、各校难度与建议
└── 相关通知文件/                  # 苏大及材化部推免工作实施办法、承诺书等
```

## 资料说明

| 文件/目录 | 用途 |
| :--- | :--- |
| `苏大材化推免去向汇总及建议.md` | 历年材化部推免生去向统计与逐校申请建议（含清华、北大、复旦、南大、浙大等） |
| `相关通知文件/` | 苏大推免工作实施办法、材化部推免实施细则、申请承诺书、创新创业学分认定标准 |

# 课程 PPT 汇报模板

> 本文件夹提供课程汇报与答辩用的 PPT 模板，含苏大官方模板与答辩模板。

## 内容介绍

含苏州大学官方 PPT 模板（含缩略图、原始 logo、校园美景素材）及一份答辩 PPT 模板，便于课程汇报、开题与毕业答辩使用。建议先看「PPT缩略图(先看这个)」挑选样式。

## 文件夹架构

```
课程PPT汇报模板/
├── 苏州大学-曹甄强-答辩PPT模板.pptx        # 答辩 PPT 模板
└── 苏州大学PPT模板/
    ├── PPT模板/                            # 苏大官方 PPT 模板文件
    ├── PPT缩略图(先看这个)/                # 各模板缩略图，便于挑选
    ├── 原始logo/                           # 苏大原始 logo 素材
    └── 校园美景图片素材/                   # 校园风景图，可用作封面/背景
```

## 资料说明

| 文件/目录 | 用途 |
| :--- | :--- |
| `苏州大学-曹甄强-答辩PPT模板.pptx` | 答辩用 PPT 模板，结构贴合毕业答辩 |

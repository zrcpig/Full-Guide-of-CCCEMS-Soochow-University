# 就业求职简历模板

> 本文件夹提供多套求职简历模板，覆盖单页、多页与封面，可按需选用。

## 内容介绍

简历模板按「单页 / 多页 / 封面」三大类整理，单页又细分活泼、极简、简约、文艺、稳重、表格六种风格。建议先看「单页简历119套」总览，挑选适合岗位调性的模板。

## 文件夹架构

```
就业求职简历模板/
├── 【1】单页简历119套/
│   ├── 【1】活泼单页13套/
│   ├── 【2】极简单页13套/
│   ├── 【3】简约单页31套/
│   ├── 【4】文艺单页21套/
│   ├── 【5】稳重单页27套/
│   └── 【6】表格简历14套/
├── 【2】多页简历47套/
│   ├── 【1】四页简历39套/
│   └── 【2】三页简历8套/
└── 【3】封面23套自由搭配（赠送）/
    ├── 封面01.docx ~ 封面23.docx
    └── 预览图-方便查找/
```

## 资料说明

| 目录 | 用途 |
| :--- | :--- |
| `【1】单页简历119套/` | 119 套单页简历，按风格分 6 类，适合一页纸投递 |
| `【2】多页简历47套/` | 47 套多页简历（三页 / 四页），适合经历较丰富者 |
| `【3】封面23套自由搭配（赠送）/` | 23 个封面模板，可自由搭配正文使用 |

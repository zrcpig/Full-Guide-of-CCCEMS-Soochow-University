# 材化部转专业

> 本文件夹汇总苏州大学本科生转专业工作方案等相关资料。

## 内容介绍

收录苏大近年本科生转专业工作方案的通知与文件，供拟转入或转出材化部的同学了解政策与流程。本指南转专业部分亦参考了校内相关开源项目，详见首页「友情感谢」。

## 文件夹架构

```
材化部转专业/
├── 关于公布苏州大学2023年本科生转专业工作方案的通知.pdf
├── 苏州大学2022级本科生转专业工作方案（2023年）.doc
└── 苏州大学2024年本科生转专业工作方案.pdf
```
